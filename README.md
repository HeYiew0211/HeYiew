## 🛠️ 环境准备

### 1. 安装 Python
- 官网下载：https://www.python.org/downloads/
- macOS：`brew install python3`
- Windows：下载安装包，**勾选 "Add Python to PATH"**
- 验证安装：终端输入 `python3 --version`（或 `python --version`）

### 2. 安装 VS Code（推荐编辑器）
- 下载：https://code.visualstudio.com/
- 安装 Python 扩展（搜索 "Python" by Microsoft）

### 3. 运行第一行代码
```bash
# 终端输入 Python 交互模式
python3
>>> print("Hello, Python!")
Hello, Python!
>>> exit()
```

## 📖 使用方式

1. **每天开始前**：打开 `01-学习计划/` 中对应日期的文件，浏览今日目标
2. **学习过程中**：参考 `02-思维导图/` 建立知识框架
3. **学完后**：完成 `03-每日题目/` 对应目录的练习
4. **遇到困难**：先看题目中的提示，再看 `answers/` 中的参考答案

### 题目难度说明
- ⭐⭐ **必做题**：必须完成，覆盖当天核心知识
- ⭐ **选做题**：有时间和兴趣就做，用于巩固和挑战

---

## 📝 学习建议

1. **动手为主**：每个代码示例都自己敲一遍，不要复制粘贴
2. **先思考再查答案**：做不出题先想 5 分钟，再查提示，最后看答案
3. **善用 `print()`**：不确定代码在做什么？用 `print()` 打印出来看
4. **Day 3 是关键**：list 和 dict 是 Python 的灵魂，多花时间
5. **Day 7 不要跳过**：综合项目能把零散知识串成体系
