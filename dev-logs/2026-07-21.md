## 技术笔记
- 项目采用单 HTML 文件架构，CSS 内嵌在 `<style>`，JS 在 `<script>`
- Canvas API 渲染 25×25 网格，每格 20px，总画布 500×500px
- setInterval 实现回合制游戏循环，三种难度三种速度
- 方向队列机制：keydown 仅写入 `nextDirection`，`gameTick` 才应用
- 障碍物安全区：蛇起始位置曼哈顿距离 ≤2 的格子不会被障碍物占据
- Boss 随机行走：打乱方向数组后逐个尝试，避开障碍物
- 暂停时保存所有 interval ID，恢复时重新创建

### 技术笔记
- pbxproj 格式严格要求 **tab 字符**缩进，不能用空格
- Xcode 26 使用 objectVersion 77
- `GENERATE_INFOPLIST_FILE = NO` + 手动 Info.plist 避免资源复制冲突
- WKWebView 加载 bundle 内的 HTML 文件，支持键盘输入
- macOS 部署目标：macOS 26.0

### 如何运行
```bash
# 在 Xcode 中打开
open macOS/SnakeGame.xcodeproj
# 或命令行构建
cd macOS && xcodebuild -project SnakeGame.xcodeproj -scheme SnakeGame build
```
