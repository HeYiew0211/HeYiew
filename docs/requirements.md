# 贪吃蛇 — 需求文档

## 项目概述
单人贪吃蛇网页游戏，单 HTML 文件，双击在浏览器中运行。支持 Mac Safari / Chrome。

## 页面结构

### 页面 1 — 主菜单（`#page-menu`）
- 游戏标题"贪吃蛇"
- 「开始游戏」按钮 → 切换到游戏页面
- 「游戏设置」按钮 → 打开难度设置弹窗

### 页面 2 — 游戏页面（`#page-game`）
- 顶部栏：左侧显示分数，右侧齿轮图标 ⚙
- Canvas 画布：25×25 网格游戏区域（500×500px）
- 蛇吃食物变长
- 边界穿越：从左出从右入，从上出从下入

### 弹窗 3 — 难度设置（`#modal-difficulty`）
- 从主菜单的「游戏设置」打开
- 三个难度选项：简单 / 普通 / 困难
- 点选后自动关闭，选择保存

### 弹窗 4 — 死亡（`#modal-death`）
- 标题"游戏结束"
- 显示最终得分
- 「重试」→ 同难度重新开始
- 「退出」→ 返回主菜单

### 弹窗 5 — 暂停（`#modal-pause`）
- 从游戏页面齿轮图标或按 P 键打开
- 「继续」→ 恢复游戏
- 「退出」→ 返回主菜单

## 难度配置

| 难度 | 障碍物数量 | Boss 数量 | Boss 移动间隔 | 蛇移动间隔 |
|------|-----------|----------|-------------|-----------|
| 简单 | 0 | 0 | - | 150ms |
| 普通 | 8 | 1 | 300ms | 120ms |
| 困难 | 15 | 2 | 200ms | 90ms |

## 死亡条件
1. 蛇头碰到障碍物
2. 蛇头碰到 Boss
3. 蛇头碰到自己身体

## 操作方式
- ↑ ↓ ← → 方向键：控制蛇的移动
- W A S D 键：同样控制方向
- P 键：暂停 / 继续

## 非功能需求
- 无外部依赖，无需安装任何软件
- 双击 `index.html` 即可游玩
- 界面语言为中文
- 视觉风格简约、湖水绿主色调、高对比度
