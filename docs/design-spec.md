## 设计理念

## 动画规范

### 过渡
- 所有颜色/背景变化：`transition: 200ms ease`
- 按钮 hover：`transform: scale(1.05)` + `box-shadow` 出现

### 弹窗
- 入场：`opacity: 0 → 1` + `transform: scale(0.95) → scale(1)`（200ms ease-out）
- 遮罩入场：`opacity: 0 → 1`（200ms）

### Boss 脉冲
- Canvas 绘制时交替使用 `#E74C3C` 和 `#C0392B`（每 500ms 切换）
- 或在 Boss 周围绘制一圈半透明红色光晕

## 图标
- **Logo**：120×120px 内联 SVG，双圆环+蛇形 S 曲线+红色信子，`logoFloat` 浮动动画
- **齿轮图标** ⚙：Unicode U+2699，字号 28px，颜色 `#5CAC9D`
- **背景粒子**：8 个湖水绿圆点，`particleDrift` 漂移动画，各不相同的位置/大小/时长
- 用 CSS/SVG 设置样式，不使用图片
