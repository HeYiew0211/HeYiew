# 每日知识点导图

> **怎么使用这些导图？**
>
> - 每天学习前先看一遍当天导图，建立整体认知框架。
> - 学习过程中用导图做导航，学完一个节点勾一个。
> - 每天结束后对着导图自问自答，检验记忆效果。
> - 每个导图独立成图，可按天查阅，不必一次全看完。

## Day 6：异常处理与推导式

```mermaid
mindmap
  Day 6 异常处理与推导式
    异常处理
      try...except
        try 包裹可能出错的代码
        except 捕获指定异常
        多个 except 分支
        except Exception as e
      finally
        无论是否异常都执行
        常用于释放资源
      else
        try 无异常时执行
      常见异常
        ValueError 值错误
        TypeError 类型错误
        KeyError 键不存在
        IndexError 索引越界
        FileNotFoundError 文件未找到
        ZeroDivisionError 除以零
      raise
        手动抛出异常
        raise ValueError("原因")
      自定义异常
        class MyError(Exception)
    list comprehension
      基本语法
        [表达式 for item in 可迭代对象]
        [x*2 for x in range(10)]
      带条件过滤
        [x for x in lst if x > 0]
      嵌套
        [(x, y) for x in a for y in b]
    dict comprehension
      基本语法
        {k: v for item in seq}
        {x: x**2 for x in range(5)}
      带条件
        {k: v for k, v in d.items() if v > 0}
      键值互换
        {v: k for k, v in d.items()}
    面向对象入门
      类与对象
        class 类名:
        类名用 PascalCase
      构造方法
        def __init__(self, ...):
        self.属性 = 参数
      self
        代表实例本身
        必须显式写
      实例方法
        def method(self, ...):
        必须带 self 参数
      创建实例
        obj = ClassName(args)
        obj.method()
```
