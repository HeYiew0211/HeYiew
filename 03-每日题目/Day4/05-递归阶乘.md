# 05 - 递归阶乘 ⭐

## 题目描述

编写一个**递归函数** `factorial(n)`，计算 n 的阶乘（n!）。

阶乘定义：
- 0! = 1
- n! = n × (n-1) × (n-2) × ... × 2 × 1（n > 0）

要求：
- 使用递归方式实现（函数调用自身）
- 然后再写一个**循环版本** `factorial_loop(n)` 做对比
- 测试 n = 5, n = 0, n = 10

## 输入输出示例

```python
print(factorial(5))        # 输出: 120（递归版）
print(factorial_loop(5))   # 输出: 120（循环版）

print(factorial(0))        # 输出: 1
print(factorial(10))       # 输出: 3628800
```

## 解题思路

> 💡 提示：
> - 递归的两个要素：**基准条件**（`n == 0` 时返回 1）和**递归调用**（`n * factorial(n-1)`）
> - 递归就像俄罗斯套娃，一层层拆开，最终回到基准条件
> - 循环版本用 `for i in range(1, n+1)` 累乘即可
> - 思考：递归和循环各有什么优缺点？
