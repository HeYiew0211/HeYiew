# 04 - 猜拳游戏 ⭐⭐

## 题目描述

使用 Python 的 `random` 模块实现一个**剪刀石头布**游戏。

要求：
- 用户输入自己的选择："剪刀"、"石头"或"布"
- 电脑用 `random.choice()` 随机出拳
- 用函数 `judge(player, computer)` 封装判断逻辑，返回 "你赢了！"、"你输了！" 或 "平局！"
- 游戏规则：剪刀赢布，布赢石头，石头赢剪刀

## 输入输出示例

```
===== 剪刀石头布游戏 =====
请输入你的选择（剪刀/石头/布）：石头
你出了：石头
电脑出了：剪刀
结果：你赢了！
```

```
请输入你的选择（剪刀/石头/布）：剪刀
你出了：剪刀
电脑出了：石头
结果：你输了！
```

## 解题思路

> 💡 提示：
> - 先 `import random`，用 `random.choice(["剪刀", "石头", "布"])` 让电脑随机出拳
> - `judge` 函数先判断平局（player == computer），再判断玩家赢的情况（三种），其余就是输
> - 可以用一个 `if` 链来处理所有胜负组合
> - 考虑用户输入大小写问题，可以用 `.strip()` 去除前后空格
