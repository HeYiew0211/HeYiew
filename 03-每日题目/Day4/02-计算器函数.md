# 02 - 计算器函数 ⭐⭐

## 题目描述

编写四个函数，实现一个简单的计算器：

- `add(a, b)` — 返回 a + b
- `subtract(a, b)` — 返回 a - b
- `multiply(a, b)` — 返回 a * b
- `divide(a, b)` — 返回 a / b，**如果除数为 0，返回错误提示字符串 "错误：除数不能为零"**

然后编写一个简单的测试，调用这四个函数并打印结果。

## 输入输出示例

```python
# 测试
print(add(10, 5))        # 输出: 15
print(subtract(10, 5))   # 输出: 5
print(multiply(10, 5))   # 输出: 50
print(divide(10, 5))     # 输出: 2.0
print(divide(10, 0))     # 输出: 错误：除数不能为零
```

## 解题思路

> 💡 提示：
> - 前三个函数非常简单，各一行代码即可
> - `divide` 函数中，先用 `if b == 0:` 判断除数为零的情况
> - 测试部分可以把所有函数的调用放在一起，方便观察结果
