# 06 - 日期计算 ⭐

## 题目描述

使用 Python 的 `datetime` 模块完成以下两个任务：

### 任务 A：计算日期差

编写函数 `days_between(date1_str, date2_str)`，接收两个日期字符串（格式 `"YYYY-MM-DD"`），返回它们之间相差的**天数**（取绝对值，始终返回正数）。

### 任务 B：N 天后的日期

编写函数 `days_later(date_str, n)`，接收一个日期字符串（格式 `"YYYY-MM-DD"`）和一个整数 n，返回 n 天后的日期字符串（同样格式）。

## 输入输出示例

```python
# === 任务 A ===
print(days_between("2024-01-01", "2024-01-10"))  # 输出: 9
print(days_between("2024-12-31", "2025-01-01"))  # 输出: 1
print(days_between("2024-06-01", "2024-06-01"))  # 输出: 0

# === 任务 B ===
print(days_later("2024-01-01", 30))   # 输出: 2024-01-31
print(days_later("2024-12-31", 1))    # 输出: 2025-01-01
print(days_later("2024-02-28", 1))    # 输出: 2024-02-29（2024是闰年）
```

## 解题思路

> 💡 提示：
> - 使用 `datetime.strptime(date_str, "%Y-%m-%d").date()` 将字符串转为 date 对象
> - 两个 date 对象相减得到一个 `timedelta` 对象，用 `.days` 获取天数
> - 取绝对值用 `abs()`
> - N 天后用 `date + timedelta(days=n)`，然后用 `.strftime("%Y-%m-%d")` 格式化回字符串
