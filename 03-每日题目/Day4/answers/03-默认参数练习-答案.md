# 03 - 默认参数练习 - 答案

## 完整代码

```python
# ========== 练习 A：多语言问候 ==========
def greet(name, greeting="你好"):
    """向指定的人打招呼
    
    参数:
        name: 姓名（字符串）
        greeting: 问候语（字符串），默认为"你好"
    """
    return f"{greeting}，{name}！"


# ========== 练习 B：灵活次方 ==========
def power(base, exp=2):
    """计算 base 的 exp 次方
    
    参数:
        base: 底数
        exp: 指数，默认为 2（计算平方）
    """
    return base ** exp


# ========== 测试 ==========
if __name__ == "__main__":
    # 测试练习 A
    print("=== 练习 A：多语言问候 ===")
    print(greet("小明"))                     # 你好，小明！
    print(greet("Alice", "Hello"))          # Hello，Alice！
    print(greet("太郎", "こんにちは"))       # こんにちは，太郎！
    print(greet("Tom", "Good morning"))     # Good morning，Tom！
    
    # 测试练习 B
    print("\n=== 练习 B：灵活次方 ===")
    print(f"5² = {power(5)}")               # 25
    print(f"2¹⁰ = {power(2, 10)}")          # 1024
    print(f"3³ = {power(3, 3)}")            # 27
    print(f"10⁰ = {power(10, 0)}")          # 1（任何数的0次方都是1）
    print(f"2⁰·⁵ = {power(2, 0.5)}")        # 1.414...（开平方）
```

## 关键要点

1. **默认参数语法**：`def func(param=default_value)` — 等号两边不加空格（PEP 8 规范）
2. **参数顺序**：有默认值的参数必须放在没有默认值的参数**后面**，否则会报语法错误
3. **f-string 格式化**：`f"{变量}"` 是 Python 3.6+ 最推荐的字符串格式化方式
4. **`**` 运算符**：`base ** exp` 是 Python 中的幂运算，比 `math.pow()` 更直观
5. **默认参数的实际用途**：让函数在大多数场景下只需传最少的参数，同时保留灵活性
