# 04 - 猜拳游戏 - 答案

## 完整代码

```python
import random


def judge(player, computer):
    """判断剪刀石头布的胜负
    
    参数:
        player: 玩家的选择（"剪刀"/"石头"/"布"）
        computer: 电脑的选择
    
    返回:
        str: "你赢了！" / "你输了！" / "平局！"
    """
    if player == computer:
        return "平局！"
    
    # 玩家赢的三种情况
    if (player == "剪刀" and computer == "布") or \
       (player == "石头" and computer == "剪刀") or \
       (player == "布" and computer == "石头"):
        return "你赢了！"
    
    # 其余都是电脑赢
    return "你输了！"


def play_game():
    """运行一轮猜拳游戏"""
    print("===== 剪刀石头布游戏 =====")
    
    # 用户输入
    player_choice = input("请输入你的选择（剪刀/石头/布）：").strip()
    
    # 输入校验
    valid_choices = ["剪刀", "石头", "布"]
    if player_choice not in valid_choices:
        print("输入无效，请输入：剪刀、石头或布")
        return
    
    # 电脑随机出拳
    computer_choice = random.choice(valid_choices)
    
    # 显示结果
    print(f"你出了：{player_choice}")
    print(f"电脑出了：{computer_choice}")
    print(f"结果：{judge(player_choice, computer_choice)}")


# ========== 运行游戏 ==========
if __name__ == "__main__":
    play_game()
```

## 关键要点

1. **`random.choice()`**：从列表中随机选取一个元素，非常适合此类场景
2. **判断逻辑的简化**：先判断平局，再判断玩家赢的三种情况，剩下的都是输 — 不需要写 9 种组合
3. **续行符 `\`**：当一行 `if` 条件太长时，用 `\` 换行，或用括号包裹
4. **输入校验**：用 `.strip()` 去除前后空格，用 `in` 检查输入是否合法
5. **函数分离**：`judge` 只负责判断逻辑，`play_game` 负责交互 — 各司其职，便于测试和扩展

### 扩展思路

如果想支持多轮游戏（直到用户输入"退出"），可以把 `play_game()` 放在一个 `while True` 循环中。
