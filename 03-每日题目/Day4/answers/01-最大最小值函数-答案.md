# 01 - 最大最小值函数 - 答案

## 完整代码

```python
def find_max_min(numbers):
    """返回列表中的最大值和最小值（不使用内置函数）
    
    参数:
        numbers: 数字列表
    
    返回:
        tuple: (最大值, 最小值)，空列表返回 (None, None)
    """
    # 处理空列表
    if len(numbers) == 0:
        return (None, None)
    
    # 初始化：将第一个元素同时设为最大值和最小值
    max_val = numbers[0]
    min_val = numbers[0]
    
    # 从第二个元素开始遍历
    for num in numbers[1:]:
        if num > max_val:
            max_val = num
        if num < min_val:
            min_val = num
    
    return (max_val, min_val)


# ========== 测试 ==========
if __name__ == "__main__":
    # 测试 1
    result = find_max_min([3, 1, 7, 4, 9, 2])
    print(f"测试1: {result}")  # (9, 1)
    
    # 测试 2：负数
    result = find_max_min([-5, -2, -10, 0])
    print(f"测试2: {result}")  # (0, -10)
    
    # 测试 3：空列表
    result = find_max_min([])
    print(f"测试3: {result}")  # (None, None)
    
    # 测试 4：单个元素
    result = find_max_min([42])
    print(f"测试4: {result}")  # (42, 42)
```

## 关键要点

1. **空列表判断**：先检查 `len(numbers) == 0`，避免访问 `numbers[0]` 时报错
2. **初始化技巧**：将第一个元素同时作为最大值和最小值的初始值，然后从第二个元素开始比较
3. **一次遍历**：在同一个 `for` 循环中同时更新最大值和最小值，效率为 O(n)
4. **切片**：`numbers[1:]` 表示从索引 1 到末尾的子列表，可以安全地跳过第一个元素
