# 06 - 日期计算 - 答案

## 完整代码

```python
from datetime import datetime, timedelta


def days_between(date1_str, date2_str):
    """计算两个日期之间相差的天数（取绝对值）
    
    参数:
        date1_str: 日期字符串，格式 "YYYY-MM-DD"
        date2_str: 日期字符串，格式 "YYYY-MM-DD"
    
    返回:
        int: 相差的天数
    """
    # 将字符串转为 date 对象
    date1 = datetime.strptime(date1_str, "%Y-%m-%d").date()
    date2 = datetime.strptime(date2_str, "%Y-%m-%d").date()
    
    # 相减得到 timedelta 对象，取 .days 属性
    delta = date2 - date1
    return abs(delta.days)


def days_later(date_str, n):
    """计算某个日期 N 天后的日期
    
    参数:
        date_str: 日期字符串，格式 "YYYY-MM-DD"
        n: 天数（可为负数，表示 N 天前）
    
    返回:
        str: N 天后的日期字符串，格式 "YYYY-MM-DD"
    """
    # 字符串转为 date 对象
    date_obj = datetime.strptime(date_str, "%Y-%m-%d").date()
    
    # 加上 timedelta
    new_date = date_obj + timedelta(days=n)
    
    # 格式化回字符串
    return new_date.strftime("%Y-%m-%d")


# ========== 测试 ==========
if __name__ == "__main__":
    # 测试任务 A：日期差
    print("=== 任务 A：计算日期差 ===")
    print(f"2024-01-01 到 2024-01-10：相差 {days_between('2024-01-01', '2024-01-10')} 天")
    print(f"2024-12-31 到 2025-01-01：相差 {days_between('2024-12-31', '2025-01-01')} 天")
    print(f"2024-06-01 到 2024-06-01：相差 {days_between('2024-06-01', '2024-06-01')} 天")
    
    # 测试任务 B：N 天后
    print("\n=== 任务 B：N 天后的日期 ===")
    print(f"2024-01-01 的 30 天后：{days_later('2024-01-01', 30)}")
    print(f"2024-12-31 的 1 天后：{days_later('2024-12-31', 1)}")
    print(f"2024-02-28 的 1 天后：{days_later('2024-02-28', 1)}")
    
    # 额外测试：N 天前（负数）
    print(f"\n2024-06-01 的 100 天前：{days_later('2024-06-01', -100)}")
    
    # 实用示例：计算距离春节还有多少天
    print("\n=== 实用示例 ===")
    today = datetime.now().date().strftime("%Y-%m-%d")
    spring_festival = "2027-02-06"  # 2027年春节
    days_left = days_between(today, spring_festival)
    print(f"今天：{today}")
    print(f"距离 2027 年春节还有 {days_left} 天")
```

## 关键要点

1. **`strptime` 和 `strftime`**：
   - `strptime`（string parse time）：字符串 → datetime 对象（解析）
   - `strftime`（string format time）：datetime 对象 → 字符串（格式化）
2. **格式代码**：`%Y`（四位年份）、`%m`（两位月份）、`%d`（两位日期）
3. **`timedelta`**：表示时间差的类型，可以加减天数、小时、分钟等
4. **`abs()`**：取绝对值，确保日期差始终为正数
5. **闰年自动处理**：`datetime` 模块自动考虑了闰年，不需要手动判断
