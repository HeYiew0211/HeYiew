# 02 - 计算器函数 - 答案

## 完整代码

```python
def add(a, b):
    """加法"""
    return a + b


def subtract(a, b):
    """减法"""
    return a - b


def multiply(a, b):
    """乘法"""
    return a * b


def divide(a, b):
    """除法，除数为 0 时返回错误提示"""
    if b == 0:
        return "错误：除数不能为零"
    return a / b


# ========== 测试 ==========
if __name__ == "__main__":
    print("===== 计算器测试 =====")
    
    print(f"10 + 5 = {add(10, 5)}")          # 15
    print(f"10 - 5 = {subtract(10, 5)}")     # 5
    print(f"10 × 5 = {multiply(10, 5)}")     # 50
    print(f"10 ÷ 5 = {divide(10, 5)}")       # 2.0
    print(f"10 ÷ 0 = {divide(10, 0)}")       # 错误：除数不能为零
    
    # 更多测试
    print(f"\n3.5 + 2.1 = {add(3.5, 2.1)}")          # 5.6
    print(f"100 - 37 = {subtract(100, 37)}")          # 63
    print(f"-3 × 6 = {multiply(-3, 6)}")              # -18
    print(f"7 ÷ 2 = {divide(7, 2)}")                  # 3.5
```

## 关键要点

1. **除零判断**：在 `divide` 函数中使用 `if b == 0` 提前返回错误信息，避免程序崩溃
2. **函数职责单一**：每个函数只做一件事，代码清晰易维护
3. **整数除法的差异**：Python 3 中 `/` 总是返回浮点数（如 `7/2 = 3.5`），如果想得到整数商可以用 `//`
4. **提前返回（early return）**：在检测到错误条件时立即 `return`，后面的正常逻辑不需要缩进在 `else` 里
