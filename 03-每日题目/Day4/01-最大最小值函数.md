# 01 - 最大最小值函数 ⭐⭐

## 题目描述

编写一个函数 `find_max_min(numbers)`，接收一个数字列表作为参数，返回其中的**最大值**和**最小值**。

要求：
- 函数返回一个 tuple `(max_value, min_value)`
- **不使用** Python 内置的 `max()` 和 `min()` 函数，自己用循环遍历实现
- 如果列表为空，返回 `(None, None)`

## 输入输出示例

```python
# 示例 1
result = find_max_min([3, 1, 7, 4, 9, 2])
print(result)  # 输出: (9, 1)

# 示例 2
result = find_max_min([-5, -2, -10, 0])
print(result)  # 输出: (0, -10)

# 示例 3
result = find_max_min([])
print(result)  # 输出: (None, None)

# 示例 4
result = find_max_min([42])
print(result)  # 输出: (42, 42)
```

## 解题思路

> 💡 提示：
> - 可以先将 `max_val` 和 `min_val` 初始化设为列表的第一个元素，然后从第二个元素开始遍历比较
> - 注意空列表的特殊情况，要先判断
> - 遍历时用 `if` 条件比较当前元素与当前最大值/最小值，如果更大就更新 max，更小就更新 min
