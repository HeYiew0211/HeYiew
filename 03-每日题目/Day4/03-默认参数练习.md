# 03 - 默认参数练习 ⭐⭐

## 题目描述

Python 函数支持默认参数，让函数调用更加灵活。请完成以下两个练习：

### 练习 A：多语言问候函数

编写函数 `greet(name, greeting="你好")`：
- 默认用中文打招呼，格式为 `"{greeting}，{name}！"`
- 也可以传入英文问候语，如 `greet("Alice", "Hello")` 输出 `"Hello，Alice！"`

### 练习 B：灵活次方函数

编写函数 `power(base, exp=2)`：
- 默认计算平方（exp=2）
- 也可以传入任意指数，如 `power(2, 10)` 计算 2 的 10 次方

## 输入输出示例

```python
# === 练习 A ===
print(greet("小明"))               # 输出: 你好，小明！
print(greet("Alice", "Hello"))     # 输出: Hello，Alice！
print(greet("太郎", "こんにちは"))  # 输出: こんにちは，太郎！

# === 练习 B ===
print(power(5))        # 输出: 25（5²）
print(power(2, 10))    # 输出: 1024（2¹⁰）
print(power(3, 3))     # 输出: 27（3³）
```

## 解题思路

> 💡 提示：
> - 默认参数在函数定义时写在 `参数名=默认值` 的格式
> - 有默认值的参数必须放在没有默认值的参数**后面**
> - `greet` 函数使用 f-string 格式化输出字符串
> - `power` 函数直接用 `base ** exp` 即可计算次方
