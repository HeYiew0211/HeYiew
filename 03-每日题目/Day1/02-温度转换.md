# 02 - 温度转换 ⭐⭐

## 题目描述

编写一个程序，让用户输入摄氏温度（Celsius），将其转换为华氏温度（Fahrenheit）并输出。

**转换公式：**

```
F = C × 9/5 + 32
```

**要求：**
- 使用 `input()` 获取用户输入的摄氏温度
- 注意 `input()` 返回的是字符串，需要用 `float()` 转换为浮点数
- 输出结果保留一位小数

## 输入输出示例

**示例 1：**

```
请输入摄氏温度：0
华氏温度：32.0
```

**示例 2：**

```
请输入摄氏温度：100
华氏温度：212.0
```

**示例 3：**

```
请输入摄氏温度：36.5
华氏温度：97.7
```

## 提示

> 💡 提示：`input()` 返回的是字符串类型（str），需要用 `float()` 转为浮点数才能进行数学运算。保留一位小数可以在 f-string 中使用格式说明符：`f"{变量:.1f}"`，其中 `.1f` 表示保留一位小数。
