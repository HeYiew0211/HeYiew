# 06 - 变量交换 ⭐

## 题目描述

有两个变量 `a` 和 `b`，分别存储了两个不同的值。请在不使用第三个变量的情况下，交换这两个变量的值。

**要求：**
- 不可以使用第三个临时变量
- 充分利用 Python 的特性

## 输入输出示例

程序不需要输入，直接演示交换过程。

**示例输出：**

```
交换前：a = 10, b = 20
交换后：a = 20, b = 10
```

## 提示

> 💡 提示：Python 有一种非常简洁的交换方式：`a, b = b, a`，一行搞定！这利用了 Python 的"元组打包与解包"特性。如果你学的是其他语言（如 C/Java），通常需要一个临时变量，但 Python 不需要。
