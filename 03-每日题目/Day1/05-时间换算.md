# 05 - 时间换算 ⭐

## 题目描述

编写一个程序，让用户输入一个总秒数（整数），将其换算为"X小时Y分钟Z秒"的格式并输出。

**换算关系：**
- 1 小时 = 3600 秒
- 1 分钟 = 60 秒

**要求：**
- 使用整除 `//` 和取余 `%` 运算符
- 输出格式为"X小时Y分钟Z秒"

## 输入输出示例

**示例 1：**

```
请输入总秒数：3661
1小时1分钟1秒
```

**示例 2：**

```
请输入总秒数：7200
2小时0分钟0秒
```

**示例 3：**

```
请输入总秒数：65
0小时1分钟5秒
```

## 提示

> 💡 提示：分三步走：
> 1. 小时 = 总秒数 // 3600
> 2. 剩余秒数 = 总秒数 % 3600，然后 分钟 = 剩余秒数 // 60
> 3. 秒 = 剩余秒数 % 60
