# 03 - 圆的面积与周长 ⭐⭐

## 题目描述

编写一个程序，让用户输入圆的半径（可以是小数），计算并输出该圆的面积和周长。

**公式：**
- 面积 = π × r²
- 周长 = 2 × π × r

**要求：**
- π 取 3.14159
- 使用 f-string 格式化输出
- 面积和周长各保留两位小数

## 输入输出示例

**示例 1：**

```
请输入圆的半径：5
面积：78.54
周长：31.42
```

**示例 2：**

```
请输入圆的半径：3.5
面积：38.48
周长：21.99
```

## 提示

> 💡 提示：半径的平方可以用 `r ** 2` 或 `r * r` 来计算。`**` 是 Python 中的幂运算符。输出两位小数用 `f"{变量:.2f}"`。
