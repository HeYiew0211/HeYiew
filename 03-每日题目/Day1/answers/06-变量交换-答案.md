# 06 - 变量交换 答案

```python
# 初始值
a = 10
b = 20

print(f"交换前：a = {a}, b = {b}")

# Python 一行交换法
a, b = b, a

print(f"交换后：a = {a}, b = {b}")
```

**运行结果：**

```
交换前：a = 10, b = 20
交换后：a = 20, b = 10
```

## 知识点

- **元组打包与解包**：`a, b = b, a` 本质上做了两件事：
  1. 右边 `b, a` 打包成一个元组 `(20, 10)`
  2. 左边 `a, b` 将元组解包，分别赋值给 a 和 b

- 这是 Python 特有的语法糖，其他语言通常需要用第三个临时变量
