# 04 - 数字拆解 答案

```python
# 获取用户输入的三位整数
num = int(input("请输入一个三位整数："))

# 拆解每一位数字
hundreds = num // 100           # 百位：整除 100
tens = (num // 10) % 10         # 十位：先整除 10 再去余
ones = num % 10                 # 个位：直接取余 10

# 输出结果
print(f"百位：{hundreds}")
print(f"十位：{tens}")
print(f"个位：{ones}")
```

**运行示例：**

```
请输入一个三位整数：357
百位：3
十位：5
个位：7
```

## 知识点

- **`//` 整除运算符**：只保留商的整数部分，如 `357 // 100 = 3`
- **`%` 取余运算符**：返回除法的余数，如 `357 % 10 = 7`
- **int()**：将字符串转换为整数
