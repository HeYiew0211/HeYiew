# 02 - 温度转换 答案

```python
# 获取用户输入的摄氏温度
celsius = float(input("请输入摄氏温度："))

# 转换为华氏温度
fahrenheit = celsius * 9 / 5 + 32

# 输出结果，保留一位小数
print(f"华氏温度：{fahrenheit:.1f}")
```

**运行示例：**

```
请输入摄氏温度：36.5
华氏温度：97.7
```

## 知识点

- **input()**：从键盘获取用户输入，返回值永远是字符串（str）
- **float()**：将字符串转换为浮点数（小数）
- **格式化输出**：`f"{变量:.1f}"` 中的 `.1f` 表示保留 1 位小数
