# 03 - 圆的面积与周长 答案

```python
# 定义 π 的值
PI = 3.14159

# 获取用户输入的半径
r = float(input("请输入圆的半径："))

# 计算面积和周长
area = PI * (r ** 2)          # 面积 = π × r²
perimeter = 2 * PI * r        # 周长 = 2 × π × r

# 输出结果，保留两位小数
print(f"面积：{area:.2f}")
print(f"周长：{perimeter:.2f}")
```

**运行示例：**

```
请输入圆的半径：5
面积：78.54
周长：31.42
```

## 知识点

- **幂运算**：`r ** 2` 表示 r 的平方（r 的 2 次方）
- **常量命名**：通常用全大写命名常量（如 `PI`），表示这个值在整个程序中不会改变
- **浮点数精度**：`.2f` 四舍五入保留两位小数
