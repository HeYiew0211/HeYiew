# 01 - Hello World：自我介绍 答案

```python
# 定义个人信息变量
name = "小明"
age = 20
hobby = "打篮球"

# 使用 f-string 组合输出
print(f"大家好，我叫{name}，今年{age}岁，我喜欢{hobby}。")
```

**运行结果：**

```
大家好，我叫小明，今年20岁，我喜欢打篮球。
```

## 知识点

- **变量**：用 `name = "小明"` 这样的方式把数据存起来
- **f-string**：`f"..."` 是格式化字符串，花括号 `{}` 里的变量会被替换成它的值
- **print()**：Python 内置函数，把内容输出到屏幕
