# 05 - 时间换算 答案

```python
# 获取用户输入的总秒数
total_seconds = int(input("请输入总秒数："))

# 计算小时、分钟、秒
hours = total_seconds // 3600              # 小时数
remaining = total_seconds % 3600           # 去掉小时后剩余的秒数
minutes = remaining // 60                  # 分钟数
seconds = remaining % 60                   # 最终的秒数

# 输出结果
print(f"{hours}小时{minutes}分钟{seconds}秒")
```

**运行示例：**

```
请输入总秒数：3661
1小时1分钟1秒
```

## 知识点

- 从大到小逐层拆解：先算最大单位（小时），余数再拆成次大单位（分钟），最后剩下最小的单位（秒）
- `//` 整除和 `%` 取余是拆解数字的黄金搭档
