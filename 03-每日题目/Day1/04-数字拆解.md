# 04 - 数字拆解 ⭐⭐

## 题目描述

编写一个程序，让用户输入一个**三位整数**，分别输出它的百位、十位、个位数字。

**要求：**
- 使用 `//`（整除）和 `%`（取余）运算符
- 不使用字符串切片的方式（即不要用 `str(num)[0]` 这种取巧方法）

## 输入输出示例

**示例 1：**

```
请输入一个三位整数：357
百位：3
十位：5
个位：7
```

**示例 2：**

```
请输入一个三位整数：108
百位：1
十位：0
个位：8
```

## 提示

> 💡 提示：对于一个三位数 `n`：
> - 百位 = `n // 100`（整除 100）
> - 十位 = `(n // 10) % 10`（先整除 10 得到前两位，再取余 10 得到十位）
> - 个位 = `n % 10`（取余 10）
>
> `//` 是整除，结果只保留整数部分（如 `357 // 100 = 3`）。`%` 是取余，返回除法后的余数（如 `357 % 10 = 7`）。
