# 01 - Hello World：自我介绍 ⭐⭐

## 题目描述

编写一个 Python 程序，用 `print()` 打印一句话来介绍你自己。

**要求：**
- 使用 f-string（格式化字符串）组合输出
- 内容至少包含：姓名、年龄、爱好
- 将姓名、年龄、爱好分别存储在变量中

## 输入输出示例

程序不需要输入，直接输出即可。

**示例输出：**

```
大家好，我叫小明，今年 20 岁，我喜欢打篮球。
```

## 提示

> 💡 提示：先用变量存储你的信息，然后用 `print(f"大家好，我叫{name}，今年{age}岁，我喜欢{hobby}。")` 这样的 f-string 格式输出。f-string 是 Python 3.6+ 的特性，用 `f"..."` 开头，花括号 `{}` 内可以直接写变量名。
