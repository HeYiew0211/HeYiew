# 04 - 通讯录雏形 ⭐⭐

## 题目描述

编写一个程序，用 **list + dict** 的组合来实现一个简易通讯录。

**数据结构：**
- 用一个 list 存储所有联系人
- 每个联系人是一个 dict，包含 `name`（姓名）和 `phone`（电话）两个字段

**功能要求：**
1. **添加联系人**：接收姓名和电话号码，添加到通讯录中
2. **按姓名查找电话**：输入姓名，在通讯录中查找对应的电话（未找到则提示"联系人不存在"）
3. **列出所有联系人**：遍历并打印通讯录中所有人的姓名和电话

**要求：**
- 写一段演示代码来展示以上 3 个功能
- 用 for 循环遍历 list 中的 dict

## 输入输出示例

程序不需要用户交互输入，直接运行演示即可。

**示例输出：**

```
=== 简易通讯录 ===

添加联系人：张三（13800138000）
添加联系人：李四（13900139000）
添加联系人：王五（13700137000）

查找"李四"的电话：13900139000
查找"赵六"的电话：联系人"赵六"不存在

所有联系人：
  张三：13800138000
  李四：13900139000
  王五：13700137000
```

## 提示

> 💡 提示：
> - 通讯录结构：`contacts = []`，每个元素是 `{"name": "张三", "phone": "138..."}` 这样的 dict
> - 添加联系人：`contacts.append({"name": name, "phone": phone})`
> - 遍历查找：用 `for contact in contacts:` 遍历，然后 `if contact["name"] == 目标姓名:` 来判断
> - 打印所有联系人：直接遍历 `for contact in contacts:` 然后取 `contact["name"]` 和 `contact["phone"]`
