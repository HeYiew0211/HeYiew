# 05 - 词频统计 ⭐

## 题目描述

给定一段英文文本，统计每个单词出现的次数，用 `dict` 存储统计结果，最后打印出现次数最多的前 3 个单词及其次数。

**给定文本：**
```
hello world hello python hello world python programming hello python code
```

**要求：**
- 用 `split()` 方法将字符串按空格拆分为单词列表
- 用 dict 统计每个单词出现的次数
- 用 `sorted()` 按出现次数降序排序，取前 3 名
- 输出格式整洁

## 输入输出示例

程序不需要用户输入。

**示例输出：**

```
文本：hello world hello python hello world python programming hello python code

词频统计结果（前 3 名）：
  hello：4 次
  python：3 次
  world：2 次
```

## 提示

> 💡 提示：
> - `text.split()` 按空格将字符串拆分成单词列表
> - 统计计数思路：遍历单词列表，如果单词已经在 dict 中，计数 +1；否则初始化为 1。可以用 `dict.get(word, 0)` 简化
> - 排序词频：`sorted(word_count.items(), key=lambda x: x[1], reverse=True)`，其中 `key=lambda x: x[1]` 表示按值（出现次数）排序，`reverse=True` 表示降序
