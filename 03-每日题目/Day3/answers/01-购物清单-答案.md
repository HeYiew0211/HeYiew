# 01 - 购物清单 答案

```python
# 初始化购物清单
shopping_list = ["苹果", "香蕉", "牛奶"]
print("=== 购物清单演示 ===")
print(f"初始清单：{shopping_list}")

# 1. 添加商品
shopping_list.append("面包")
print(f"\n添加"面包"后：{shopping_list}")

# 2. 删除商品
shopping_list.remove("香蕉")
print(f"删除"香蕉"后：{shopping_list}")

# 3. 查看商品数量
print(f"当前商品总数：{len(shopping_list)}")

# 4. 尝试删除一个不存在的商品
print(f"\n尝试删除"西瓜"...（不存在）")
item_to_remove = "西瓜"
if item_to_remove in shopping_list:
    shopping_list.remove(item_to_remove)
else:
    print(f"商品"{item_to_remove}"不在清单中！")
```

**运行结果：**

```
=== 购物清单演示 ===
初始清单：['苹果', '香蕉', '牛奶']

添加"面包"后：['苹果', '香蕉', '牛奶', '面包']
删除"香蕉"后：['苹果', '牛奶', '面包']
当前商品总数：3

尝试删除"西瓜"...（不存在）
商品"西瓜"不在清单中！
```

## 知识点

- **list（列表）**：Python 中最常用的数据结构，用 `[]` 定义，元素有序、可修改
- **append()**：在列表末尾追加元素
- **remove()**：删除指定元素（只删第一个匹配项）
- **len()**：获取列表长度
- **in 运算符**：判断元素是否在列表中，安全地检查后再删除，避免报错
