# 04 - 通讯录雏形 答案

```python
# 初始化空的通讯录列表
contacts = []

print("=== 简易通讯录 ===")

# 1. 添加联系人
contacts.append({"name": "张三", "phone": "13800138000"})
print(f"\n添加联系人：张三（13800138000）")

contacts.append({"name": "李四", "phone": "13900139000"})
print(f"添加联系人：李四（13900139000）")

contacts.append({"name": "王五", "phone": "13700137000"})
print(f"添加联系人：王五（13700137000）")

# 2. 按姓名查找电话的函数式写法
def find_phone(contacts, target_name):
    """在通讯录中查找指定姓名的电话"""
    for contact in contacts:
        if contact["name"] == target_name:
            return contact["phone"]
    return None  # 没找到返回 None

# 查找演示
print(f"\n查找"李四"的电话：{find_phone(contacts, "李四")}")

result = find_phone(contacts, "赵六")
if result:
    print(f"查找"赵六"的电话：{result}")
else:
    print(f"查找"赵六"的电话：联系人"赵六"不存在")

# 3. 列出所有联系人
print(f"\n所有联系人：")
for contact in contacts:
    print(f"  {contact["name"]}：{contact["phone"]}")
```

**运行结果：**

```

添加联系人：张三（13800138000）
添加联系人：李四（13900139000）
添加联系人：王五（13700137000）

查找"李四"的电话：13900139000
查找"赵六"的电话：联系人"赵六"不存在

所有联系人：
  张三：13800138000
  李四：13900139000
  王五：13700137000
```

## 知识点

- **list + dict 组合**：这是 Python 中非常常见的数据组织方式。list 负责"多个"，dict 负责"属性"
- **遍历查找模式**：用 for 循环遍历 list，检查每个 dict 的某个字段是否匹配目标
- **函数封装**：将查找逻辑封装成函数，返回结果，让代码更清晰
