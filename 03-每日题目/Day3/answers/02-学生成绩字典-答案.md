# 02 - 学生成绩字典 答案

```python
# 1. 初始化：创建包含 5 个学生的字典
scores = {
    "小明": 92,
    "小华": 78,
    "小丽": 85,
    "小刚": 66,
    "小美": 95,
}

print("=== 学生成绩管理系统 ===")

# 2. 安全查询：使用 get() 方法
print(f"\n查询小明的成绩：{scores.get("小明", "学生不存在")}")
print(f"查询小红的成绩：{scores.get("小红", "学生小红不存在")}")

# 3. 添加新学生
scores["小红"] = 88
print(f"\n添加新学生小红（88分）")

# 4. 打印所有学生成绩
print(f"\n所有学生成绩：")
for name, score in scores.items():
    print(f"  {name}：{score}")

# 5. 计算平均分
average = sum(scores.values()) / len(scores)
print(f"\n全班平均分：{average:.1f}")
```

**运行结果：**

```
=== 学生成绩管理系统 ===

查询小明的成绩：92
查询小红的成绩：学生小红不存在

添加新学生小红（88分）

所有学生成绩：
  小明：92
  小华：78
  小丽：85
  小刚：66
  小美：95
  小红：88

全班平均分：84.0
```

## 知识点

- **dict（字典）**：键值对（key-value）结构，用 `{}` 定义，通过 key 快速查找 value
- **get(key, default)**：安全查询，key 不存在时返回 default 而非报错
- **直接赋值添加**：`dict["新key"] = value` 即可添加新条目
- **items()**：返回所有键值对，用于 `for name, score in scores.items()` 遍历
- **values()**：返回所有值，配合 `sum()` 和 `len()` 计算平均
