# 03 - 列表去重排序 答案

```python
# 原始列表（包含重复元素）
nums = [3, 1, 2, 3, 1, 4, 2, 5]
print(f"原始列表：{nums}")

# 去重 + 排序：一行搞定
result = sorted(set(nums))
print(f"去重排序后：{result}")
```

**运行结果：**

```
原始列表：[3, 1, 2, 3, 1, 4, 2, 5]
去重排序后：[1, 2, 3, 4, 5]
```

## 知识点

- **set（集合）**：特点是无序、元素不重复。`set(nums)` 将列表转为集合，自动去除重复元素
- **sorted()**：返回一个新的排序列表。`sorted(set(nums))` 先转为集合去重，再排序
- 这行代码展示了 Python 链式处理的优雅：从内到外执行，`set()` → `sorted()`
