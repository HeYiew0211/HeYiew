# 05 - 词频统计 答案

```python
# 给定的英文文本
text = "hello world hello python hello world python programming hello python code"
print(f"文本：{text}")

# 1. 用 split() 拆分为单词列表
words = text.split()

# 2. 用 dict 统计每个单词的出现次数
word_count = {}
for word in words:
    # 用 get() 获取当前次数，不存在则返回 0，然后 +1
    word_count[word] = word_count.get(word, 0) + 1

# 3. 按出现次数降序排序，取前 3 名
# items() 返回 (单词, 次数) 的元组列表
# key=lambda x: x[1] 表示按次数排序
# reverse=True 表示降序（从高到低）
sorted_words = sorted(word_count.items(), key=lambda x: x[1], reverse=True)
top_3 = sorted_words[:3]  # 取前 3 个

# 4. 打印结果
print(f"\n词频统计结果（前 3 名）：")
for word, count in top_3:
    print(f"  {word}：{count} 次")
```

**运行结果：**

```
文本：hello world hello python hello world python programming hello python code

词频统计结果（前 3 名）：
  hello：4 次
  python：3 次
  world：2 次
```

## 知识点

- **split()**：按空格将字符串拆成列表，如 `"a b c".split()` → `['a', 'b', 'c']`
- **dict 计数模式**：`dict[key] = dict.get(key, 0) + 1` 是经典的计数写法
- **sorted() 高级用法**：`key=lambda x: x[1]` 指定按元组的第二个元素（即次数）排序
- **切片取前 N 个**：`list[:3]` 取列表中的前 3 个元素
