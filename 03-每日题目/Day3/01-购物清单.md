# 01 - 购物清单 ⭐⭐

## 题目描述

编写一个程序，用 `list`（列表）来管理一个购物清单，支持以下操作并通过模拟演示来展示功能：

1. **添加商品**：向清单中添加一个商品名称
2. **删除商品**：从清单中删除指定商品（如果商品不存在，给出提示）
3. **查看所有商品**：打印当前清单中的所有商品
4. **查看商品数量**：打印当前清单中的商品总数

**要求：**
- 使用 list 存储商品名称
- 使用 `append()`、`remove()`、`len()` 等列表方法
- 写一段模拟代码来演示所有功能

## 输入输出示例

程序不需要用户交互输入，直接运行演示即可。

**示例输出：**

```
=== 购物清单演示 ===
初始清单：['苹果', '香蕉', '牛奶']

添加"面包"后：['苹果', '香蕉', '牛奶', '面包']
删除"香蕉"后：['苹果', '牛奶', '面包']
当前商品总数：3

尝试删除"西瓜"...（不存在）
商品"西瓜"不在清单中！
```

## 提示

> 💡 提示：
> - `list.append(item)` — 在列表末尾添加元素
> - `list.remove(item)` — 删除列表中第一个匹配的元素（如果不存在会报错，可以用 `if item in list` 先判断）
> - `len(list)` — 返回列表的长度（元素个数）
> - `item in list` — 判断某个元素是否在列表中，返回 True 或 False
