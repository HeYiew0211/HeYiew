# 02 - 学生成绩字典 ⭐⭐

## 题目描述

编写一个程序，使用 `dict`（字典）来管理学生姓名和成绩。实现以下功能：

1. **初始化**：创建一个包含 5 个学生的字典（姓名→成绩）
2. **查询成绩**：查询某个学生的成绩，使用 `get()` 方法安全查询（如果学生不存在，给出友好提示）
3. **添加新学生**：添加一个新的学生及其成绩
4. **打印所有学生**：遍历字典，打印所有学生的姓名和成绩
5. **计算平均分**：计算并打印所有学生的平均分

**要求：**
- 使用 dict 存储数据
- 安全查询用 `get()` 而非直接 `dict[key]`
- 遍历字典用 `for` 循环
- 平均分保留一位小数

## 输入输出示例

程序不需要用户交互输入，直接运行演示即可。

**示例输出：**

```
=== 学生成绩管理系统 ===

查询小明的成绩：92
查询小红的成绩：学生小红不存在

添加新学生小红（88分）

所有学生成绩：
  小明：92
  小华：78
  小丽：85
  小刚：66
  小美：95
  小红：88

全班平均分：84.0
```

## 提示

> 💡 提示：
> - `dict[key]` 在 key 不存在时会报错（KeyError），而 `dict.get(key, 默认值)` 在 key 不存在时返回默认值（不报错），如 `scores.get("小明", "学生不存在")`
> - 遍历字典的所有键值对：`for name, score in scores.items():`
> - 计算平均分：`sum(scores.values()) / len(scores)`
