# 03 - 列表去重排序 ⭐⭐

## 题目描述

给定一个包含重复元素的列表，要求去重并按从小到大排序。

**给定列表：**
```python
nums = [3, 1, 2, 3, 1, 4, 2, 5]
```

**要求：**
- 利用 `set`（集合）进行去重
- 使用 `sorted()` 或 `list.sort()` 进行排序
- 打印去重排序前后的列表

## 输入输出示例

程序不需要用户输入。

**示例输出：**

```
原始列表：[3, 1, 2, 3, 1, 4, 2, 5]
去重排序后：[1, 2, 3, 4, 5]
```

## 提示

> 💡 提示：
> - `set` 是 Python 中的集合类型，特点是无序且元素不重复。将一个 list 传给 `set()` 即可去重：`set(nums)`
> - 但 `set` 是无序的，需要再转回 list 并排序。最简洁的写法是：`sorted(set(nums))`
> - `sorted()` 返回一个新的排序后的列表，原列表不变；`list.sort()` 是原地排序，会修改原列表
