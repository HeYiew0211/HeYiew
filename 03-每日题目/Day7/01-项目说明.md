# 01 - 命令行通讯录管理器 ⭐⭐ 必做

## 项目名称

命令行通讯录管理器

## 项目目标

综合运用 Day1~Day6 所学知识（变量、数据类型、条件判断、循环、函数、文件读写、异常处理、类与对象），实现一个功能完整的命令行通讯录管理工具。

## 功能需求

1. **添加联系人** — 输入姓名、电话、邮箱，保存到通讯录
2. **查看所有联系人** — 以格式化表格打印所有联系人
3. **搜索联系人** — 按姓名模糊匹配（支持部分姓名搜索）
4. **删除联系人** — 按序号或姓名删除指定联系人
5. **数据持久化** — 程序退出时自动保存到 JSON 文件；启动时自动加载已有数据

## 数据结构设计引导

用 **list** 存储所有联系人，每个联系人是 **dict**：

```python
# 通讯录整体是一个 list
contacts = [
    {"name": "张三", "phone": "13800138000", "email": "zhangsan@example.com"},
    {"name": "李四", "phone": "13900139000", "email": "lisi@example.com"},
]
```

每个联系人 dict 包含三个字段：
| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | str | 联系人姓名 |
| `phone` | str | 电话号码 |
| `email` | str | 电子邮箱 |

## 函数设计引导

请设计并实现以下函数：

| 函数 | 功能 | 关键点 |
|------|------|--------|
| `load_data()` | 从 JSON 文件加载通讯录数据 | `json.load()` + try/except 处理文件不存在 |
| `save_data(contacts)` | 将通讯录数据保存到 JSON 文件 | `json.dump()` + `ensure_ascii=False` + `indent=2` |
| `add_contact(contacts)` | 添加新联系人 | `input()` 获取信息 → 构造 dict → `list.append()` |
| `view_all(contacts)` | 显示所有联系人 | `for` 循环 + f-string 格式化输出 |
| `search_contact(contacts)` | 按姓名搜索联系人 | 遍历 + `in` 关键字做模糊匹配 |
| `delete_contact(contacts)` | 删除联系人 | 先查找 → 确认 → `list.pop(index)` |

## 菜单设计引导

使用 `while True` 循环实现主菜单：

```
===== 命令行通讯录管理器 =====
1. 添加联系人
2. 查看所有联系人
3. 搜索联系人
4. 删除联系人
5. 退出
==============================
请选择操作（1-5）：
```

- 选项 1~4 分别调用对应函数
- 选项 5 调用 `save_data()` 保存数据，然后 `break` 退出循环
- `if/elif/else` 分支结构处理不同选项
