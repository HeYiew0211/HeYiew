# 04 - 猜数字游戏 答案

```python
# 预设秘密数字
secret = 42
count = 0  # 猜测次数计数器

print("我已经想好了一个数字，来猜猜看吧！")

while True:
    # 获取用户猜测
    guess = int(input("请输入你的猜测："))
    count += 1  # 每猜一次，计数器加 1

    # 判断并给出提示
    if guess > secret:
        print("大了")
    elif guess < secret:
        print("小了")
    else:
        print(f"恭喜你猜对了！你一共猜了 {count} 次。")
        break  # 猜对后跳出循环，结束游戏
```

**运行示例：**

```
我已经想好了一个数字，来猜猜看吧！

请输入你的猜测：50
大了

请输入你的猜测：25
小了

请输入你的猜测：42
恭喜你猜对了！你一共猜了 3 次。
```

## 知识点

- **while True**：无限循环，直到遇到 `break` 才退出
- **break**：立即跳出当前循环
- **计数器模式**：`count += 1` 是 `count = count + 1` 的简写，用于统计循环执行了多少次
