# 01 - 成绩等级 答案

```python
# 获取用户输入的分数
score = float(input("请输入分数（0-100）："))

# 判断分数等级
if score > 100 or score < 0:
    print("分数不合法！")
elif score >= 90:
    print("等级：A")
elif score >= 80:
    print("等级：B")
elif score >= 70:
    print("等级：C")
elif score >= 60:
    print("等级：D")
else:
    print("等级：F")
```

**运行示例：**

```
请输入分数（0-100）：95
等级：A
```

## 知识点

- **if-elif-else 结构**：从上到下依次判断，命中即停止
- **条件顺序**：从大到小排列，避免冗余条件（走到 `elif score >= 80` 时已经确定 `< 90`）
- **边界检查**：先用 `if` 判断输入是否在有效范围内
