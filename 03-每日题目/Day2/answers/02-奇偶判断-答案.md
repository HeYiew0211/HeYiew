# 02 - 奇偶判断 答案

```python
# 获取用户输入的整数
num = int(input("请输入一个整数："))

# 判断奇偶
if num % 2 == 0:
    print(f"{num} 是偶数")
else:
    print(f"{num} 是奇数")
```

**运行示例：**

```
请输入一个整数：7
7 是奇数
```

## 知识点

- **取余运算符 `%`**：`num % 2` 返回 num 除以 2 的余数
  - 余数为 0 → 偶数
  - 余数为 1 → 奇数
- **if-else**：二选一结构，条件成立走 if 分支，否则走 else 分支
