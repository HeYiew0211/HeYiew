# 03 - 打印 9×9 乘法表 答案

```python
# 外层循环：控制行（乘数 i 从 1 到 9）
for i in range(1, 10):
    # 内层循环：控制列（被乘数 j 从 1 到 i）
    for j in range(1, i + 1):
        # 打印算式，end="\t" 表示用 Tab 分隔而不换行
        print(f"{i}×{j}={i * j}", end="\t")
    # 内层循环结束后换行
    print()
```

## 知识点

- **嵌套 for 循环**：外层每执行一次，内层完整执行一轮
- **range(1, 10)**：生成 1 到 9 的整数序列（左闭右开）
- **print() 的 end 参数**：默认 `end="\n"`（换行），改为 `end="\t"` 则不换行，用 Tab 分隔
- **空 print()**：输出一个换行，用于内层循环结束后换到下一行
