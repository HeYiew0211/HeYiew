# 06 - 素数判断 答案

```python
# 获取用户输入
n = int(input("请输入一个正整数："))

# 判断是否为素数
if n <= 1:
    print(f"{n} 不是素数")
else:
    is_prime = True  # 先假设是素数
    # 只需要检查 2 到 √n
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:       # 找到了一个因数
            is_prime = False
            break            # 不用继续找了
    # 根据标记输出结果
    if is_prime:
        print(f"{n} 是素数")
    else:
        print(f"{n} 不是素数")
```

**运行示例：**

```
请输入一个正整数：17
17 是素数
```

## 知识点

- **标记变量（flag）模式**：用 `is_prime = True` 先假设是素数，一旦找到因数就设为 `False`
- **`break`**：找到第一个因数后立即退出循环，节省计算
- **`n ** 0.5`**：求平方根（0.5 次方），`int()` 取整数部分
- **为什么只检查到 √n**：因数成对出现，如果 ≤√n 范围内没有因数，那么 >√n 的范围也不会有
