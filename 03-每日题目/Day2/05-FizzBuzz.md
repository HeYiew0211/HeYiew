# 05 - FizzBuzz ⭐

## 题目描述

编写程序打印 1 到 100 的数字，但遵循以下规则：

- 如果是 3 的倍数，打印 **"Fizz"**（不打印数字）
- 如果是 5 的倍数，打印 **"Buzz"**（不打印数字）
- 如果同时是 3 和 5 的倍数，打印 **"FizzBuzz"**（不打印数字）
- 否则，打印数字本身

**要求：**
- 使用 `for` 循环遍历 1 到 100
- 使用 `if-elif-else` 判断

**注意：** 这是经典面试题，考察条件判断的顺序。

## 输入输出示例

程序不需要输入。

**示例输出（前 20 个）：**

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
...
```

## 提示

> 💡 提示：**条件顺序非常关键！** 如果先判断 `num % 3 == 0`，那么遇到 15 时，会先命中这个条件打印 "Fizz" 而不是正确的 "FizzBuzz"。正确的做法是把"同时是 3 和 5 的倍数"这个条件放在最前面判断：
>
> ```python
> if num % 3 == 0 and num % 5 == 0:
>     print("FizzBuzz")
> elif num % 3 == 0:
>     print("Fizz")
> elif num % 5 == 0:
>     print("Buzz")
> else:
>     print(num)
> ```
>
> 另一种思路：同时是 3 和 5 的倍数也就是 15 的倍数（3 和 5 的最小公倍数），所以也可以用 `num % 15 == 0` 判断。
