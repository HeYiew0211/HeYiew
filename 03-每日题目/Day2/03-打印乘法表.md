# 03 - 打印 9×9 乘法表 ⭐⭐

## 题目描述

编写一个程序，用 `for` 循环打印出标准的 9×9 乘法表，格式整齐。

**要求：**
- 使用两个嵌套的 `for` 循环，`range(1, 10)` 从 1 到 9
- 每个算式占一个位置，用 `\t`（Tab）分隔
- 每行换行打印

## 输入输出示例

程序不需要输入，输出如下格式：

```
1×1=1
2×1=2   2×2=4
3×1=3   3×2=6   3×3=9
4×1=4   4×2=8   4×3=12  4×4=16
5×1=5   5×2=10  5×3=15  5×4=20  5×5=25
6×1=6   6×2=12  6×3=18  6×4=24  6×5=30  6×6=36
7×1=7   7×2=14  7×3=21  7×4=28  7×5=35  7×6=42  7×7=49
8×1=8   8×2=16  8×3=24  8×4=32  8×5=40  8×6=48  8×7=56  8×8=64
9×1=9   9×2=18  9×3=27  9×4=36  9×5=45  9×6=54  9×7=63  9×8=72  9×9=81
```

## 提示

> 💡 提示：外层循环控制行（乘数 `i` 从 1 到 9），内层循环控制列（被乘数 `j` 从 1 到 `i`）。
>
> ```python
> for i in range(1, 10):       # 行：1~9
>     for j in range(1, i+1):  # 列：1~i
>         print(f"{i}×{j}={i*j}", end="\t")
>     print()  # 换行
> ```
>
> `print()` 默认会在末尾加换行。用 `end="\t"` 可以把默认的换行替换成 Tab，实现在同一行打印多个算式。内层循环结束后调用一个空 `print()` 来换行。
