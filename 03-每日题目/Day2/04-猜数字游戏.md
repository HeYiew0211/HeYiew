# 04 - 猜数字游戏 ⭐⭐

## 题目描述

编写一个猜数字游戏程序。

**规则：**
- 程序预设一个秘密数字（比如 `42`）
- 用户不断输入猜测的数字
- 每次猜测后，程序给出提示："大了"、"小了"或"恭喜你猜对了！"
- 猜对后显示用户一共猜了多少次

**要求：**
- 使用 `while` 循环
- 使用 `if-elif-else` 判断
- 用一个计数器记录猜测次数

## 输入输出示例

```
我已经想好了一个数字，来猜猜看吧！

请输入你的猜测：50
大了

请输入你的猜测：25
小了

请输入你的猜测：42
恭喜你猜对了！你一共猜了 3 次。
```

## 提示

> 💡 提示：整体结构如下：
> ```python
> secret = 42
> count = 0
> while True:
>     guess = int(input("请输入你的猜测："))
>     count += 1
>     if guess > secret:
>         print("大了")
>     elif guess < secret:
>         print("小了")
>     else:
>         print(f"恭喜你猜对了！你一共猜了 {count} 次。")
>         break  # 猜对就跳出循环
> ```
>
> `while True:` 是无限循环，必须配合 `break` 语句在条件满足时跳出，否则程序永远不会停止。
