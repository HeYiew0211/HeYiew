# 06 - 素数判断 ⭐

## 题目描述

编写一个程序，让用户输入一个正整数，判断它是否为素数（质数）。

**什么是素数？**
- 素数（质数）是指大于 1 的自然数中，除了 1 和它本身以外不再有其他因数的数
- 例如：2、3、5、7、11、13、17、19 都是素数
- 1 不是素数

**要求：**
- 用 `for` 循环检查是否存在因数
- 优化：只需要检查 2 到 √n 之间的数即可（提示：`int(n ** 0.5)` 可得到 √n 的整数部分）

## 输入输出示例

**示例 1：**

```
请输入一个正整数：17
17 是素数
```

**示例 2：**

```
请输入一个正整数：100
100 不是素数
```

**示例 3：**

```
请输入一个正整数：1
1 不是素数
```

## 提示

> 💡 提示：判断一个数 n 是否为素数：
> 1. 如果 n <= 1，不是素数
> 2. 从 2 到 √n（`int(n ** 0.5)`）遍历，如果 n 能被其中任何一个数整除，则不是素数
> 3. 如果遍历完都没有找到因数，则是素数
>
> 为什么只需要检查到 √n？因为如果 n 有一个大于 √n 的因数，那么必然有一个小于 √n 的因数与之配对。所以如果 2~√n 之间没有因数，就不用再往上找了。
