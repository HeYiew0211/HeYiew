# 01 - 成绩等级 ⭐⭐

## 题目描述

编写一个程序，让用户输入一个 0-100 的分数，根据以下规则输出对应的等级：

| 分数范围 | 等级 |
|---------|------|
| >= 90   | A    |
| >= 80 且 < 90 | B |
| >= 70 且 < 80 | C |
| >= 60 且 < 70 | D |
| < 60    | F    |

**要求：**
- 使用 `if-elif-else` 结构
- 如果输入的分数不在 0-100 范围内，提示"分数不合法"

## 输入输出示例

**示例 1：**

```
请输入分数（0-100）：95
等级：A
```

**示例 2：**

```
请输入分数（0-100）：72
等级：C
```

**示例 3：**

```
请输入分数（0-100）：55
等级：F
```

**示例 4：**

```
请输入分数（0-100）：120
分数不合法！
```

## 提示

> 💡 提示：`if-elif-else` 是从上到下依次判断的，一旦命中某个条件就跳出。所以可以从最高分开始写条件，顺序很重要！比如先判断 `>= 90`，再 `elif >= 80`，依此类推，不需要写 `and < 90`，因为能走到第二个 `elif` 说明已经小于 90 了。
