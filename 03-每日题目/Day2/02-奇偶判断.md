# 02 - 奇偶判断 ⭐⭐

## 题目描述

编写一个程序，让用户输入一个整数，使用 `if-else` 判断它是奇数还是偶数。

**判断方法：**
- 如果一个数除以 2 的余数为 0，则为偶数，否则为奇数

**要求：**
- 使用 `%`（取余）运算符
- 使用 `if-else` 结构

## 输入输出示例

**示例 1：**

```
请输入一个整数：7
7 是奇数
```

**示例 2：**

```
请输入一个整数：100
100 是偶数
```

## 提示

> 💡 提示：`num % 2 == 0` 表示 num 是偶数，因为偶数除以 2 没有余数。`%` 是取余运算符。
