# 03 - 列表推导式转换 ⭐⭐

## 题目描述

用 **list comprehension（列表推导式）** 完成以下三个任务，并写出等价的**传统 for 循环写法**做对比。

### 任务 A：字符串转大写

给定一个字符串列表 `["hello", "world", "python", "code"]`，生成一个新列表，其中所有字符串转为大写。

### 任务 B：过滤数字

生成 1 到 100 中**所有能被 3 或 5 整除的数**的列表。

### 任务 C：生成随机数列表

生成一个包含 **20 个 1 到 100 之间随机整数**的列表（使用 `random` 模块）。

## 输入输出示例

```python
# 任务 A
words = ["hello", "world", "python", "code"]
# 列表推导式版本
upper_words = [???]
print(upper_words)  # ['HELLO', 'WORLD', 'PYTHON', 'CODE']

# 任务 B
# 输出: [3, 5, 6, 9, 10, 12, 15, ...]
# 总共有多少个？应该是 47 个

# 任务 C
# 输出类似: [42, 17, 89, 3, 65, ...]（20个随机数）
```

## 解题思路

> 💡 提示：
> - 列表推导式基本语法：`[表达式 for 变量 in 可迭代对象 if 条件]`
> - 任务 A：`[word.upper() for word in words]`
> - 任务 B：`[n for n in range(1, 101) if n % 3 == 0 or n % 5 == 0]`
> - 任务 C：先 `import random`，然后 `[random.randint(1, 100) for _ in range(20)]`
> - `%` 是取模运算符（求余数），`n % 3 == 0` 表示 n 能被 3 整除
> - `_` 作为变量名表示"我不关心这个值"，纯用于循环计数
