# 06 - 输入验证装饰器 ⭐

## 题目描述

编写一个通用的"持续要求输入直到合法"的模式。

### 任务

编写函数 `get_int(prompt)`，持续要求用户输入一个整数，直到输入合法为止。如果用户输入的不是整数（如字母、小数），捕获异常并提示重新输入。

要求：
- 使用 `while True` + `try/except ValueError` 模式
- 输入合法后返回该整数
- 提示信息清晰友好

## 输入输出示例

```
请输入一个整数：abc
输入无效，请输入一个整数！

请输入一个整数：3.14
输入无效，请输入一个整数！

请输入一个整数：42
你输入的整数是：42
```

### 进阶

将这个模式扩展为一个通用的 `get_input(prompt, converter)` 函数，其中 `converter` 是一个转换函数（如 `int`、`float`），这样可以同时用于要求整数和浮点数的场景。

```python
# 用法
age = get_input("请输入年龄：", int)
score = get_input("请输入分数：", float)
```

## 解题思路

> 💡 提示：
> - `try` 块中调用 `int(input(...))`，如果用户输入非数字，会抛出 `ValueError`
> - `except ValueError:` 捕获异常，打印提示后循环继续
> - 不要在 `except` 块中 `return`，因为出错时需要继续循环
> - 进阶版中，`converter` 作为参数传入，`try` 块中调用 `converter(user_input)`
> - 通用版还可以加入 `validator` 参数做进一步验证（如范围检查）
