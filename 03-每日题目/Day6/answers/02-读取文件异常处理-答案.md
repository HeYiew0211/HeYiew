# 02 - 读取文件异常处理 - 答案

## 完整代码

```python
def read_file_with_retry():
    """尝试读取用户指定的文件，最多 3 次机会"""
    
    max_attempts = 3
    attempt = 0
    
    while attempt < max_attempts:
        file_path = input("请输入文件名：").strip()
        attempt += 1
        
        try:
            with open(file_path, "r", encoding="utf-8") as f:
                content = f.read()
            
            # 读取成功
            print(f"\n===== 文件内容 ({file_path}) =====")
            print(content)
            return  # 成功则结束函数
        
        except FileNotFoundError:
            remaining = max_attempts - attempt
            if remaining > 0:
                print(f"文件不存在，请重试（剩余 {remaining} 次机会）\n")
            else:
                print("已超过最大尝试次数（3次），程序退出。")
        
        except PermissionError:
            remaining = max_attempts - attempt
            if remaining > 0:
                print(f"没有权限读取该文件，请重试（剩余 {remaining} 次机会）\n")
            else:
                print("已超过最大尝试次数（3次），程序退出。")
        
        except Exception as e:
            remaining = max_attempts - attempt
            if remaining > 0:
                print(f"读取文件时出错：{e}，请重试（剩余 {remaining} 次机会）\n")
            else:
                print("已超过最大尝试次数（3次），程序退出。")


# ========== 运行 ==========
if __name__ == "__main__":
    read_file_with_retry()


# ========== 常见文件异常类型 ==========
"""
FileNotFoundError  — 文件不存在
PermissionError    — 没有读取/写入权限
IsADirectoryError  — 路径是一个目录而非文件
UnicodeDecodeError — 编码错误（如用 utf-8 读 GBK 编码的文件）
OSError            — 其他操作系统相关错误
"""
```

## 关键要点

1. **重试模式**：`while` 循环 + 计数器 + `try/except` 是处理可重试错误的经典模式
2. **剩余次数提示**：`remaining = max_attempts - attempt` 给用户友好的反馈
3. **提前返回**：读取成功后用 `return` 直接结束函数，避免执行后面的代码
4. **多重 except**：可以捕获多种异常，给用户不同的提示
5. **通用兜底**：`except Exception as e:` 捕获所有未预料的异常，`e` 是异常对象，包含具体错误信息
6. **`.strip()`**：去除用户输入的前后空白，避免因不小心输了个空格导致文件找不到
