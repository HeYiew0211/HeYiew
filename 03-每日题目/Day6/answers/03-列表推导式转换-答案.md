# 03 - 列表推导式转换 - 答案

## 完整代码

```python
import random

# ============================================================
# 任务 A：字符串转大写
# ============================================================
words = ["hello", "world", "python", "code"]

# 列表推导式版本（一行代码！）
upper_words = [word.upper() for word in words]

# 传统 for 循环版本（对比）
upper_words_loop = []
for word in words:
    upper_words_loop.append(word.upper())

print("=== 任务 A：字符串转大写 ===")
print(f"推导式版本：{upper_words}")
print(f"循环版本：  {upper_words_loop}")


# ============================================================
# 任务 B：过滤 1-100 中能被 3 或 5 整除的数
# ============================================================

# 列表推导式版本
divisible = [n for n in range(1, 101) if n % 3 == 0 or n % 5 == 0]

# 传统 for 循环版本
divisible_loop = []
for n in range(1, 101):
    if n % 3 == 0 or n % 5 == 0:
        divisible_loop.append(n)

print("\n=== 任务 B：能被 3 或 5 整除的数 ===")
print(f"共有 {len(divisible)} 个数")
print(f"前 10 个：{divisible[:10]}")
print(f"后 10 个：{divisible[-10:]}")
print(f"推导式和循环版本一致：{divisible == divisible_loop}")


# ============================================================
# 任务 C：生成 20 个 1-100 随机数
# ============================================================

# 列表推导式版本
# _ 是 Python 惯例：表示"我不关心这个循环变量的值"
random_nums = [random.randint(1, 100) for _ in range(20)]

# 传统 for 循环版本
random_nums_loop = []
for _ in range(20):
    random_nums_loop.append(random.randint(1, 100))

print("\n=== 任务 C：随机数列表 ===")
print(f"随机数列表：{random_nums}")


# ============================================================
# 列表推导式语法总结
# ============================================================
"""
基本语法：
    [表达式 for 变量 in 可迭代对象]
    
带条件过滤：
    [表达式 for 变量 in 可迭代对象 if 条件]

带 if-else 表达式：
    [表达式1 if 条件 else 表达式2 for 变量 in 可迭代对象]

示例：
    # 平方数
    [x**2 for x in range(10)]  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
    
    # 只取偶数
    [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]
    
    # 偶数变正，奇数变负
    [x if x % 2 == 0 else -x for x in range(5)]  # [0, -1, 2, -3, 4]
"""
```

## 关键要点

1. **列表推导式 = 声明式编程**：描述"你要什么"而不是"怎么做"，代码更紧凑、可读性更高
2. **`_` 变量名**：Python 惯例，表示这个变量不会被使用，仅用于循环计数
3. **`%` 取模运算符**：`n % 3 == 0` 表示 n 能被 3 整除（余数为 0）
4. **推导式 vs 循环**：
   - 推导式更短、更快（Python 内部优化过）
   - 但太复杂的推导式会降低可读性，此时建议用传统 for 循环
5. **条件位置**：`if` 在**后面**是过滤（`[x for x in data if condition]`），`if-else` 在**前面**是映射（`[a if cond else b for x in data]`）
