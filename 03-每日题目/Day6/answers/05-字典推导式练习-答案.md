# 05 - 字典推导式练习 - 答案

## 任务 A：单词长度字典

### 代码

```python
# 给定的单词列表
words = ["apple", "banana", "cherry"]

# 字典推导式：{单词: 单词长度 for 单词 in 列表}
word_lengths = {word: len(word) for word in words}

print(word_lengths)
```

### 运行结果

```
{'apple': 5, 'banana': 6, 'cherry': 6}
```

### 解释

字典推导式的语法为 `{key_expr: value_expr for item in iterable}`：
- `key_expr`：`word` — 以单词本身作为 key
- `value_expr`：`len(word)` — 以单词长度作为 value
- `for word in words`：遍历列表中的每一个单词

## 任务 B：键值互换

### 代码

```python
# 给定的字典
original = {"a": 1, "b": 2, "c": 3}

# 字典推导式：{原value: 原key for key, value in 原字典.items()}
swapped = {v: k for k, v in original.items()}

print(swapped)
```

### 运行结果

```
{1: 'a', 2: 'b', 3: 'c'}
```

### 解释

- `.items()` 方法返回一个包含 `(key, value)` 元组的可迭代对象
- `for k, v in original.items()` 同时解包 key 和 value
- 新字典以原 value（`v`）为 key、原 key（`k`）为 value，实现互换

---

## 完整测试代码

```python
print("========== 任务 A：单词长度字典 ==========")

words = ["apple", "banana", "cherry", "watermelon", "kiwi"]
word_lengths = {word: len(word) for word in words}
print(f"单词列表：{words}")
print(f"单词长度字典：{word_lengths}")

print("\n========== 任务 B：键值互换 ==========")

original = {"a": 1, "b": 2, "c": 3}
swapped = {v: k for k, v in original.items()}
print(f"原始字典：{original}")
print(f"互换后字典：{swapped}")

# 验证：互换两次应回到原始
swapped_back = {v: k for k, v in swapped.items()}
print(f"再次互换：{swapped_back}")
print(f"与原始一致：{swapped_back == original}")

print("\n========== 思考题：value 有重复时 ==========")

# 如果原字典的 value 有重复，交换后会发生什么？
dup_dict = {"a": 1, "b": 2, "c": 1}
dup_swapped = {v: k for k, v in dup_dict.items()}
print(f"有重复 value 的字典：{dup_dict}")
print(f"互换后：{dup_swapped}")
print("注意：key 'c' 被 key 'a' 覆盖了！因为 value 1 重复出现，后遍历到的值覆盖了先前的值。")
```

## 知识点总结

| 知识点 | 说明 |
|--------|------|
| dict comprehension | `{k: v for item in iterable}` — 一行代码生成字典，是 Python 的语法糖 |
| `.items()` | dict 的方法，返回 `dict_items` 对象，可迭代，每个元素是 `(key, value)` 元组 |
| 元组解包 | `for k, v in original.items()` 同时把元组的两个元素赋值给两个变量 |
| `len()` | 内置函数，对字符串返回字符个数，对列表返回元素个数 |
| 键唯一性 | 字典的 key 必须唯一，重复 value 作 key 时后面的会覆盖前面的 |
| 适用场景 | 字典推导式适合简单的映射转换；逻辑复杂时用传统 for 循环更清晰 |

## 更多推导式示例

```python
# 带条件的字典推导式：仅保留长度大于 5 的单词
words = ["apple", "banana", "cherry", "kiwi"]
long_words = {word: len(word) for word in words if len(word) > 5}
print(long_words)  # {'banana': 6, 'cherry': 6}

# 数值运算：生成 1~5 的平方字典
squares = {x: x ** 2 for x in range(1, 6)}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 从两个列表构造字典
keys = ["name", "age", "city"]
values = ["Alice", 25, "Beijing"]
person = {keys[i]: values[i] for i in range(len(keys))}
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'Beijing'}

# 等价于 zip 写法（推荐）
person = dict(zip(keys, values))
print(person)  # 同上
```
