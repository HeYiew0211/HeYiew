# 06 - 输入验证装饰器 - 答案

## 基础版：`get_int(prompt)`

### 代码

```python
def get_int(prompt):
    """
    持续要求用户输入一个整数，直到输入合法为止。

    参数：
        prompt: str — 提示语

    返回：
        int — 用户输入的合法整数
    """
    while True:
        user_input = input(prompt)
        try:
            return int(user_input)
        except ValueError:
            print(f"输入无效，请输入一个整数！你输入的是：{user_input}")
```

### 测试

```python
# 测试 get_int
age = get_int("请输入你的年龄：")
print(f"你输入的年龄是：{age}")
```

## 进阶版：通用 `get_input(prompt, converter)`

### 代码

```python
def get_input(prompt, converter):
    """
    通用的输入验证函数，持续要求输入直到 converter 转换成功。

    参数：
        prompt: str — 提示语
        converter: callable — 转换函数，如 int、float

    返回：
        转换后的值
    """
    while True:
        user_input = input(prompt)
        try:
            return converter(user_input)
        except ValueError:
            # 获取 converter 的名称，用于友好的错误提示
            converter_name = converter.__name__
            print(f"输入无效，请输入一个有效的 {converter_name} 类型！你输入的是：{user_input}")
```

### 测试

```python
age = get_input("请输入你的年龄：", int)
print(f"年龄：{age}")

score = get_input("请输入你的分数：", float)
print(f"分数：{score}")
```

## 高级版：带范围验证

### 代码

```python
def get_input(prompt, converter, validator=None):
    """
    通用的输入验证函数，支持范围检查。

    参数：
        prompt: str — 提示语
        converter: callable — 转换函数，如 int、float
        validator: callable or None — 验证函数，接收转换后的值，返回 True/False
                    如果为 None，则不进行额外验证

    返回：
        转换后且通过验证的值
    """
    while True:
        user_input = input(prompt)
        try:
            value = converter(user_input)
        except ValueError:
            converter_name = converter.__name__
            print(f"输入无效，请输入一个有效的 {converter_name} 类型！你输入的是：{user_input}")
            continue

        # 如果有验证器，执行验证
        if validator is None:
            return value

        if validator(value):
            return value
        else:
            print(f"输入值 {value} 不在有效范围内，请重新输入！")
```

### 测试

```python
# 年龄必须在 0 ~ 150 之间
age = get_input("请输入你的年龄（0-150）：", int, validator=lambda x: 0 <= x <= 150)
print(f"年龄：{age}")

# 分数必须在 0 ~ 100 之间
score = get_input("请输入你的分数（0-100）：", float, validator=lambda x: 0 <= x <= 100)
print(f"分数：{score}")

# 没有 validator，直接转换
name = input("请输入你的名字：")  # 字符串不需要转换
print(f"名字：{name}")
```

## 完整测试脚本

```python
def get_int(prompt):
    """持续要求输入整数直到合法"""
    while True:
        user_input = input(prompt)
        try:
            return int(user_input)
        except ValueError:
            print(f"输入无效，请输入一个整数！你输入的是：{user_input}")


def get_input(prompt, converter, validator=None):
    """通用输入验证函数，支持类型转换和范围验证"""
    while True:
        user_input = input(prompt)
        try:
            value = converter(user_input)
        except ValueError:
            converter_name = converter.__name__
            print(f"输入无效，请输入一个有效的 {converter_name} 类型！你输入的是：{user_input}")
            continue

        if validator is None:
            return value

        if validator(value):
            return value
        else:
            print(f"输入值 {value} 不在有效范围内，请重新输入！")


if __name__ == "__main__":
    print("========== 基础版测试 ==========")
    age = get_int("请输入你的年龄：")
    print(f"你输入的年龄是：{age}")

    print("\n========== 进阶版测试 ==========")
    height = get_input("请输入你的身高（cm）：", float)
    print(f"身高：{height}")

    print("\n========== 高级版测试（带范围验证）==========")
    score = get_input(
        "请输入你的考试成绩（0-100）：",
        float,
        validator=lambda x: 0 <= x <= 100
    )
    print(f"考试成绩：{score}")
```

## 知识点总结

| 知识点 | 说明 |
|--------|------|
| `while True` | 无限循环，只有在输入合法时通过 `return` 退出 |
| `try/except ValueError` | 捕获 `int()` 或 `float()` 转换失败时抛出的异常 |
| `converter.__name__` | 获取函数的名称字符串（如 `"int"`、`"float"`），用于生成友好的错误提示 |
| `continue` | 异常处理完后重新开始循环，不执行后续代码 |
| 高阶函数 | `get_input` 接收另一个函数（`converter`、`validator`）作为参数，体现了函数是一等公民的设计理念 |
| `lambda` 表达式 | `lambda x: 0 <= x <= 100` 是匿名函数，简洁地定义一个简单的验证逻辑 |
| 默认参数 | `validator=None` 使参数可选，不传时跳过验证环节 |
| 链式比较 | `0 <= x <= 100` 是 Python 特有的链式比较语法，等价于 `0 <= x and x <= 100` |

## 扩展思考

1. **支持多值输入**：修改函数使其支持一次输入多个值（如用逗号分隔），返回列表。
2. **添加重试次数上限**：给用户最多 3 次尝试机会，超过后抛出异常或返回默认值。
3. **支持正则验证**：在 `validator` 之外再增加一个 `pattern` 参数，在字符串阶段就用正则做格式校验（如邮箱格式）。
4. **装饰器方式**：将这个验证模式封装成一个装饰器，自动为被装饰函数的参数做类型和范围验证。
