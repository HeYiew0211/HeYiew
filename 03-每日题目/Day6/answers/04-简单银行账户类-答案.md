## 完整代码

```python
class BankAccount:
    """
    简单银行账户类

    属性：
        owner: str — 账户持有人姓名
        balance: float — 余额，默认 0.0

    方法：
        deposit(amount): 存款
        withdraw(amount): 取款
        show_balance(): 显示账户信息
    """

    def __init__(self, owner, balance=0.0):
        """
        构造方法，初始化账户

        参数：
            owner: str — 账户持有人姓名
            balance: float — 初始余额，默认为 0.0
        """
        self.owner = owner
        self.balance = balance

    def deposit(self, amount):
        """
        存款方法

        参数：
            amount: float — 存款金额

        要求 amount > 0，否则打印错误提示
        """
        if amount <= 0:
            print(f"金额必须大于 0！你输入的金额为：{amount}")
            return
        self.balance += amount
        print(f"存款 {amount:.2f} 元成功！当前余额：{self.balance:.2f} 元")

    def withdraw(self, amount):
        """
        取款方法

        参数：
            amount: float — 取款金额

        要求：
        1. amount > 0，否则打印错误提示
        2. 余额充足，否则打印余额不足提示
        """
        if amount <= 0:
            print(f"金额必须大于 0！你输入的金额为：{amount}")
            return
        if amount > self.balance:
            print(f"余额不足！当前余额：{self.balance:.2f} 元，无法取款 {amount:.2f} 元")
            return
        self.balance -= amount
        print(f"取款 {amount:.2f} 元成功！当前余额：{self.balance:.2f} 元")

    def show_balance(self):
        """显示当前账户信息"""
        print(f"账户：{self.owner}，余额：{self.balance:.2f} 元")


# ============================================================
# 测试代码
# ============================================================
if __name__ == "__main__":
    print("========== 测试 1：创建账户 ==========")
    alice = BankAccount("Alice", 1000.0)
    bob = BankAccount("Bob")  # 使用默认余额 0.0
    alice.show_balance()
    bob.show_balance()

    print("\n========== 测试 2：正常存款 ==========")
    alice.deposit(500)

    print("\n========== 测试 3：正常取款 ==========")
    alice.withdraw(300)

    print("\n========== 测试 4：余额不足 ==========")
    alice.withdraw(2000)

    print("\n========== 测试 5：取款金额为负数 ==========")
    alice.withdraw(-100)

    print("\n========== 测试 6：存款金额为 0 ==========")
    alice.deposit(0)

    print("\n========== 测试 7：Bob 账户操作 ==========")
    bob.deposit(200)
    bob.show_balance()
    bob.withdraw(50)
    bob.show_balance()
```

## 知识点总结

| 知识点 | 说明 |
|--------|------|
| `class` 关键字 | 用 `class BankAccount:` 定义一个类，类名通常用大驼峰命名（PascalCase） |
| `__init__(self, ...)` | 构造方法，在创建实例时自动调用，`self` 代表实例本身 |
| `self.owner = owner` | 将传入参数赋值给实例属性，每个实例拥有自己独立的属性值 |
| 实例方法 | 类中定义的方法第一个参数必须是 `self`，调用时 Python 自动传入 |
| 默认参数 | `balance=0.0` 给构造方法参数一个默认值，使得 `BankAccount("Bob")` 也可以正常工作 |
| 条件判断 | `if amount <= 0:` 做输入合法性检查，这是一种防御式编程手法 |
| `return` 提前退出 | 检查不通过时用 `return` 提前结束方法，这叫做"卫语句"（guard clause）模式 |
| `f"{value:.2f}"` | f-string 的格式化语法，`.2f` 表示保留两位小数显示 |
| `if __name__ == "__main__":` | Python 常用模式：文件被直接运行时执行测试代码，被 import 时不执行 |
| 封装思想 | 将数据和操作数据的方法封装在同一个类中，是面向对象编程的核心思想 |
