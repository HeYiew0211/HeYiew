# 01 - 安全除法 ⭐⭐

## 题目描述

编写一个函数 `safe_divide(a, b)`，用 `try/except` 处理可能出现的异常，确保程序不会崩溃。

需要处理的异常：
1. **ZeroDivisionError** — 除数为 0
2. **TypeError** — 传入的参数不是数字（如字符串）

要求：
- 除法成功时返回计算结果（浮点数）
- 发生异常时返回对应的错误提示字符串
- 测试多种情况：正常除法、除零、类型错误

## 输入输出示例

```python
print(safe_divide(10, 2))      # 输出: 5.0
print(safe_divide(10, 0))      # 输出: 错误：除数不能为零
print(safe_divide(10, "a"))    # 输出: 错误：请输入数字
print(safe_divide("hello", 2)) # 输出: 错误：请输入数字
```

## 解题思路

> 💡 提示：
> - `try` 块中写 `return a / b`
> - `except ZeroDivisionError:` 捕获除零异常
> - `except TypeError:` 捕获类型错误
> - 也可以写成一个通用的 `except Exception as e:` 来捕获所有异常，然后返回 `str(e)`
> - 思考：`except` 的顺序重要吗？如果先写 `except Exception`，后面的 `except ZeroDivisionError` 还会执行吗？
