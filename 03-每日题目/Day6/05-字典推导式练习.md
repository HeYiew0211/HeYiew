# 05 - 字典推导式练习 ⭐

## 题目描述

用 **dict comprehension（字典推导式）** 完成以下两个任务：

### 任务 A：单词长度字典

给定一个单词列表 `["apple", "banana", "cherry"]`，生成一个字典，key 为单词，value 为该单词的长度。

```python
# 期望输出
{"apple": 5, "banana": 6, "cherry": 6}
```

### 任务 B：键值互换

给定一个 dict，交换它的 key 和 value（假设所有 value 都是唯一的，可以作为新 key）。

```python
# 输入
original = {"a": 1, "b": 2, "c": 3}

# 期望输出
{1: "a", 2: "b", 3: "c"}
```

## 解题思路

> 💡 提示：
> - 字典推导式语法：`{key_expr: value_expr for item in iterable}`
> - 任务 A：`{word: len(word) for word in words}`
> - 任务 B：`{v: k for k, v in original.items()}` 
> - `.items()` 返回一个包含 (key, value) 元组的可迭代对象
> - 思考：如果原 dict 的 value 有重复，交换后会发生什么？
