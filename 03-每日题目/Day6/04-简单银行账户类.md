# 04 - 简单银行账户类 ⭐⭐

## 题目描述

定义一个 `BankAccount` 类，模拟一个简单的银行账户。

**属性：**
- `owner` — 账户持有人姓名（str）
- `balance` — 余额（float，默认 0.0）

**方法：**
- `deposit(amount)` — 存款，金额必须大于 0，成功时打印存款金额和当前余额
- `withdraw(amount)` — 取款，金额必须大于 0；余额不足时打印提示；成功时打印取款金额和当前余额
- `show_balance()` — 显示当前账户信息和余额

**要求：**
- 创建两个账户实例并做测试
- 测试存款、取款、余额不足、取款金额为负数等场景

## 输入输出示例

```python
# 创建账户
alice = BankAccount("Alice", 1000.0)
bob = BankAccount("Bob")  # 默认余额为 0

# 操作
alice.show_balance()
# 输出: 账户：Alice，余额：1000.0 元

alice.deposit(500)
# 输出: 存款 500.0 元成功！当前余额：1500.0 元

alice.withdraw(2000)
# 输出: 余额不足！当前余额：1500.0 元，无法取款 2000.0 元

alice.withdraw(300)
# 输出: 取款 300.0 元成功！当前余额：1200.0 元

alice.withdraw(-100)
# 输出: 金额必须大于 0

bob.deposit(200)
bob.show_balance()
# 输出: 账户：Bob，余额：200.0 元
```

## 解题思路

> 💡 提示：
> - 类用 `class BankAccount:` 定义
> - `__init__(self, owner, balance=0.0)` 是构造方法，`self` 代表实例本身
> - `self.owner = owner` 将参数赋值给实例属性
> - 每个方法的第一个参数必须是 `self`
> - 用 `if amount <= 0:` 判断金额合法性
> - 取款前检查 `if amount > self.balance:` 判断余额是否充足
> - `f"{self.balance:.2f}"` 可以格式化显示两位小数
