# 02 - 读取文件异常处理 ⭐⭐

## 题目描述

编写程序，让用户输入文件名，程序尝试读取并打印文件内容。如果文件不存在，提示用户重新输入，**最多给 3 次机会**。超过 3 次仍未找到文件，则退出程序。

要求：
- 用 `try/except` 捕获 `FileNotFoundError`
- 用一个计数器记录尝试次数
- 文件读取成功后，打印完整内容并结束循环

## 输入输出示例

```
请输入文件名：notexist.txt
文件不存在，请重试（剩余 2 次机会）

请输入文件名：also_not_exist.txt
文件不存在，请重试（剩余 1 次机会）

请输入文件名：hello.txt

===== 文件内容 =====
Hello World!
Welcome to Python.
```

超过 3 次：
```
请输入文件名：a.txt
文件不存在，请重试（剩余 2 次机会）

请输入文件名：b.txt
文件不存在，请重试（剩余 1 次机会）

请输入文件名：c.txt
已超过最大尝试次数（3次），程序退出。
```

## 解题思路

> 💡 提示：
> - 用 `max_attempts = 3` 设置最大尝试次数
> - `while` 循环配合 `attempt < max_attempts` 控制循环次数
> - 每次循环 `attempt += 1`
> - `except FileNotFoundError:` 中计算剩余次数 `max_attempts - attempt`
> - 文件读取成功时用 `break` 跳出循环
> - 还可以额外捕获 `PermissionError`（没权限读取）等其他异常
