# 01 - 姓名格式化 ⭐⭐

## 题目描述

编写程序，用户输入 `"姓 名"`（拼音，全小写，中间用空格分隔，如 `"zhang san"`），程序将其格式化为 **首字母大写** 的形式（`"Zhang San"`）。

要求：
- 用 `split()` 将字符串拆分为姓和名
- 用 `capitalize()` 或 `title()` 方法将每个单词首字母大写
- 用 `join()` 将处理后的单词重新拼接

## 输入输出示例

```
请输入姓名（拼音，如 zhang san）：zhang san
格式化后：Zhang San
```

```
请输入姓名（拼音，如 zhang san）：ma yun
格式化后：Ma Yun
```

```
请输入姓名（拼音，如 zhang san）：elon musk
格式化后：Elon Musk
```

## 解题思路

> 💡 提示：
> - `split()` 默认按空白字符（空格、制表符等）分割，返回一个列表
> - `str.capitalize()` 将字符串首字母大写，其余小写
> - `str.title()` 将每个单词首字母大写，但注意它对 "o'reilly" 这类单词的处理可能不符合预期
> - 用列表推导式 `[word.capitalize() for word in words]` 可以一行处理所有单词
> - `" ".join(list)` 用空格将列表元素拼接成字符串
