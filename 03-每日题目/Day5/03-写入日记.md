# 03 - 写入日记 ⭐⭐

## 题目描述

编写一个简单的命令行日记程序。程序运行时：
1. 提示用户输入日记内容
2. 自动获取当前时间戳
3. 将日记内容**追加写入** `diary.txt` 文件，格式为：

```
--- 2024-07-22 14:30:25 ---
今天学习了 Python 的文件操作，非常有趣！
```

要求：
- 使用 `datetime` 模块获取当前时间
- 使用 `with open(...)` 语句和 `'a'`（append 追加）模式
- 每篇日记之间用分割线 `---` 隔开
- 支持连续写多篇日记（输入 `q` 退出）

## 输入输出示例

```
===== 我的日记本 =====
请输入日记内容（输入 q 退出）：今天天气真好，学习了 Python 函数
日记已保存！

请输入日记内容（输入 q 退出）：下午用 random 模块写了猜拳游戏
日记已保存！

请输入日记内容（输入 q 退出）：q
再见！
```

`diary.txt` 文件中内容示例：
```
--- 2024-07-22 14:30:25 ---
今天天气真好，学习了 Python 函数

--- 2024-07-22 15:45:10 ---
下午用 random 模块写了猜拳游戏
```

## 解题思路

> 💡 提示：
> - `datetime.now().strftime("%Y-%m-%d %H:%M:%S")` 生成时间戳字符串
> - `with open("diary.txt", "a", encoding="utf-8") as f:` 以追加模式打开，指定 utf-8 编码避免中文乱码
> - `'a'` 模式：文件不存在时会自动创建，存在则在末尾追加
> - 写入时用 `f.write()` 方法，需要手动加换行符 `\n`
