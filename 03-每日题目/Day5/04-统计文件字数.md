# 04 - 统计文件字数 ⭐⭐

## 题目描述

编写程序，读取一个文本文件，统计以下三项数据：

1. **总行数** — 文件中有多少行
2. **总单词数** — 文件中有多少个单词（以空白字符分隔）
3. **总字符数** — 文件中包含多少个字符（含空格和换行符）

要求：
- 使用 `with open()` 读取文件
- 文件路径可由用户输入
- 统计完成后格式化输出结果

## 输入输出示例

假设 `sample.txt` 内容如下：
```
Hello World
Python is great
```

运行结果：
```
请输入文件路径：sample.txt

===== 文件统计结果 =====
文件名：sample.txt
总行数：2
总单词数：5
总字符数：28
```

## 解题思路

> 💡 提示：
> - 一次性读取全部内容：`content = f.read()`，然后统计字符数（`len(content)`）
> - 统计行数：`len(content.split('\n'))` 或用 `readlines()` 读取
> - 统计单词数：`len(content.split())` — `split()` 不传参数时按任意空白字符分割，非常适合分词
> - 也可以分行读取：`for line in f:`  逐行累加统计
> - 注意：使用 `encoding="utf-8"` 以正确处理中文文件
