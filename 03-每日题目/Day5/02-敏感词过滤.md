# 02 - 敏感词过滤 ⭐⭐

## 题目描述

编写一个函数 `filter_words(text, sensitive_words)`，接收一段文本和一个敏感词列表，将文本中的所有敏感词替换为 `"***"`，返回过滤后的文本。

要求：
- 遍历敏感词列表，用 `str.replace()` 逐个替换
- 替换的 `***` 长度固定为 3 个星号（不随敏感词长度变化）
- 编写测试：用一段包含广告、脏话的模拟文本进行过滤

## 输入输出示例

```python
text = "加我微信 wx12345 领取免费礼品！这个游戏太垃圾了！"
sensitive = ["微信", "wx", "免费", "垃圾"]

filtered = filter_words(text, sensitive)
print(filtered)
# 输出: 加我*** ***12345 领取***礼品！这个游戏太***了！
```

## 解题思路

> 💡 提示：
> - `str.replace(old, new)` 返回替换后的新字符串（字符串不可变，原字符串不会变）
> - 遍历敏感词列表，每次替换后更新 text 变量：`text = text.replace(word, "***")`
> - 注意替换顺序：如果敏感词列表中有 "wx" 和 "微信"，先替换长的还是短的会影响结果
> - 有多个敏感词时，可以考虑按长度从长到短排序再替换，避免短词先替换掉长词的一部分
