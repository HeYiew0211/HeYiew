# 06 - 格式化表格 ⭐

## 题目描述

编写函数 `print_table(data)`，接收一个 **list of dict** 格式的数据（如学生成绩表），用 f-string 对齐打印成一个整齐的表格。

要求：
- 第一行打印列名（从 dict 的 keys 中获取）
- 列宽自动适配：每列宽度取"列名长度"和"该列所有数据中最大长度"中的较大值
- 数据左对齐，数字也可以左对齐（简化处理）
- 用 `-` 画一条分隔线

## 输入输出示例

```python
students = [
    {"姓名": "张三", "语文": "90", "数学": "85", "英语": "92"},
    {"姓名": "李四", "语文": "78", "数学": "92", "英语": "88"},
    {"姓名": "王小明", "语文": "95", "数学": "88", "英语": "100"},
]

print_table(students)
```

输出效果：
```
姓名      语文    数学    英语
------------------------------
张三      90      85      92
李四      78      92      88
王小明    95      88      100
```

## 解题思路

> 💡 提示：
> - 先获取所有列名：`keys = list(data[0].keys())`
> - 计算每列最大宽度：遍历所有行，用 `max(len(row[key]) for row in data)` 找到该列数据的最大长度，再和列名长度比较取 max
> - 用 f-string 对齐：`f"{value:<{width}}"` — `<` 表示左对齐，`{width}` 是变量宽度
> - 打印表头时，先打印列名，再打印分隔线 `"-" * 总宽度`
> - 可以把每列的宽度存在一个 dict 中，方便复用
