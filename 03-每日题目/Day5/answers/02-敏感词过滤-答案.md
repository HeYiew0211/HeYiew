# 02 - 敏感词过滤 - 答案

## 完整代码

```python
def filter_words(text, sensitive_words):
    """将文本中的敏感词替换为 ***
    
    参数:
        text: 原始文本
        sensitive_words: 敏感词列表
    
    返回:
        str: 过滤后的文本
    """
    # 方法：按敏感词长度从长到短排序，避免短词先替换掉长词的一部分
    # 例如：先替换 "微信" 再替换 "wx"，否则 "微信" 中的 "wx" 部分会被先替换
    sorted_words = sorted(sensitive_words, key=len, reverse=True)
    
    for word in sorted_words:
        text = text.replace(word, "***")
    
    return text


# ========== 测试 ==========
if __name__ == "__main__":
    # 测试 1：基础过滤
    text1 = "加我微信 wx12345 领取免费礼品！这个游戏太垃圾了！"
    sensitive = ["微信", "wx", "免费", "垃圾"]
    print("原始文本：", text1)
    print("过滤后：", filter_words(text1, sensitive))
    
    # 测试 2：短词先替换的问题演示
    print("\n=== 排序的重要性 ===")
    text2 = "加我微信123"
    # 如果不排序，先替换 "wx" 会影响 "微信"
    bad_order = ["wx", "微信"]
    good_order = sorted(bad_order, key=len, reverse=True)
    
    result_bad = text2
    for w in bad_order:
        result_bad = result_bad.replace(w, "***")
    
    result_good = text2
    for w in good_order:
        result_good = result_good.replace(w, "***")
    
    print(f"不排序替换：{result_bad}")   # "加我***信123" — 错误
    print(f"排序后替换：{result_good}")   # "加我***123" — 正确
    
    # 测试 3：无敏感词
    text3 = "今天天气真好！"
    print(f"\n无敏感词：{filter_words(text3, sensitive)}")
```

## 运行结果

```
原始文本： 加我微信 wx12345 领取免费礼品！这个游戏太垃圾了！
过滤后： 加我*** ***12345 领取***礼品！这个游戏太***了！

=== 排序的重要性 ===
不排序替换：加我***信123
排序后替换：加我***123

无敏感词：今天天气真好！
```

## 关键要点

1. **`sorted(key=len, reverse=True)`**：按敏感词长度从长到短排序，确保长词先被替换，避免短词破坏长词
2. **`str.replace()` 返回新字符串**：Python 字符串是不可变的，`replace` 不会修改原字符串，必须接收返回值
3. **逐个替换**：`for word in sorted_words: text = text.replace(word, "***")` — 每次替换后更新 text
4. **固定长度星号**：`"***"` 长度固定为 3，不随敏感词长度变化，能更好地隐藏敏感词长度信息
