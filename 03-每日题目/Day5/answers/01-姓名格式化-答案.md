# 01 - 姓名格式化 - 答案

## 完整代码

```python
def format_name(name):
    """将拼音姓名格式化为首字母大写的形式
    
    参数:
        name: 全小写拼音姓名，如 "zhang san"
    
    返回:
        str: 格式化后的姓名，如 "Zhang San"
    """
    # 1. 用 split() 拆分成单词列表
    words = name.split()
    
    # 2. 将每个单词首字母大写
    capitalized_words = [word.capitalize() for word in words]
    
    # 3. 用空格重新拼接
    return " ".join(capitalized_words)


# ========== 主程序 ==========
if __name__ == "__main__":
    user_input = input("请输入姓名（拼音，如 zhang san）：").strip()
    result = format_name(user_input)
    print(f"格式化后：{result}")


# ========== 方法对比 ==========
# .capitalize()：整个字符串首字母大写，其余小写
#   "hello WORLD".capitalize() → "Hello world"
#
# .title()：每个单词首字母大写，其余小写
#   "hello world".title() → "Hello World"
#   但注意 .title() 对 "o'reilly" 会变成 "O'Reilly"
#
# 本题使用 .capitalize() 更安全，逐个单词处理
```

## 关键要点

1. **`split()` vs `split(' ')`**：不传参数时按**任意空白字符**分割（空格、Tab、换行等），连续空白也会合并处理，更健壮
2. **列表推导式**：`[word.capitalize() for word in words]` 是 Python 中处理列表的简洁写法
3. **`join()` 的方向**：`"分隔符".join(列表)` — 分隔符在前，列表作为参数
4. **`.capitalize()` vs `.title()`**：`.capitalize()` 只让整个字符串的第一个字符大写，`.title()` 让每个单词首字母大写。本题逐个单词调用 `.capitalize()` 最可控
5. **`.strip()`**：去除用户输入前后的空白字符，是输入处理的良好习惯
