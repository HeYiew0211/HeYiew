## 完整代码

```python
def count_file_stats(file_path):
    """统计文本文件的行数、单词数和字符数
    
    参数:
        file_path: 文件路径
    
    返回:
        dict: 包含 line_count, word_count, char_count 的字典
    """
    with open(file_path, "r", encoding="utf-8") as f:
        content = f.read()
    
    # 统计行数：按换行符分割
    # 注意：如果文件末尾有换行，split 会产生一个空字符串，需要过滤掉
    lines = content.split('\n')
    # 如果最后一行是空的（文件末尾有换行符），不算一行
    if lines and lines[-1] == '':
        lines = lines[:-1]
    line_count = len(lines)
    
    # 统计单词数：按任意空白字符分割
    # split() 不传参数会合并连续空白，非常适合分词
    words = content.split()
    word_count = len(words)
    
    # 统计字符数：直接取字符串长度（包含所有字符，含空格和换行）
    char_count = len(content)
    
    return {
        "line_count": line_count,
        "word_count": word_count,
        "char_count": char_count
    }


# ========== 主程序 ==========
if __name__ == "__main__":
    path = input("请输入文件路径：").strip()
    
    try:
        stats = count_file_stats(path)
        print("\n===== 文件统计结果 =====")
        print(f"文件名：{path}")
        print(f"总行数：{stats['line_count']}")
        print(f"总单词数：{stats['word_count']}")
        print(f"总字符数：{stats['char_count']}")
    except FileNotFoundError:
        print(f"错误：文件 '{path}' 不存在！")
    except Exception as e:
        print(f"读取文件时出错：{e}")
```

## 关键要点

1. **三种统计方式**：
   - 行数：`content.split('\n')`，注意处理末尾空行
   - 单词数：`content.split()`（不传参数），按任意空白分割且合并连续空白
   - 字符数：`len(content)`，直接统计包括所有空白字符
2. **末尾空行处理**：文件末尾的换行符会导致 `split('\n')` 产生一个空字符串，需要过滤
3. **`split()` 不传参数 vs 传参数**：
   - `split()` → 按任意空白分割，合并连续空白
   - `split(' ')` → 只按空格分割，连续空格会产生空字符串
4. **异常处理**：用 `try/except` 捕获文件不存在的错误，给出友好提示
