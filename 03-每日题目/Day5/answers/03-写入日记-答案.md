# 03 - 写入日记 - 答案

## 完整代码

```python
from datetime import datetime


def write_diary():
    """命令行日记本：将用户输入写入 diary.txt 文件"""
    print("===== 我的日记本 =====")
    print("（输入 q 退出）")
    print()
    
    # 日记文件路径（与脚本同目录）
    diary_file = "diary.txt"
    
    while True:
        # 获取用户输入
        content = input("请输入日记内容：").strip()
        
        # 退出判断
        if content.lower() == "q":
            print("再见！")
            break
        
        # 跳过空内容
        if not content:
            print("日记内容不能为空！")
            continue
        
        # 获取当前时间戳
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        
        # 追加写入文件
        # 'a' 模式：文件存在则在末尾追加，不存在则创建
        # encoding='utf-8'：确保中文正常写入
        with open(diary_file, "a", encoding="utf-8") as f:
            f.write(f"--- {timestamp} ---\n")
            f.write(content + "\n\n")  # 日记后加两个换行，与下一篇隔开
        
        print("日记已保存！\n")


# ========== 运行 ==========
if __name__ == "__main__":
    write_diary()
```

## 文件模式对照表

| 模式 | 含义 | 文件不存在时 |
|------|------|-------------|
| `'r'` | 只读 | 报错 |
| `'w'` | 写入（覆盖） | 创建新文件 |
| `'a'` | 追加（append） | 创建新文件 |
| `'x'` | 创建新文件（排他） | 创建，存在则报错 |
| `'r+'` | 读写 | 报错 |

## 关键要点

1. **`'a'` 模式**：在文件末尾追加内容，不会覆盖已有内容，是日记程序的最佳选择
2. **`encoding="utf-8"`**：指定文件编码，处理中文时必须设置，否则可能乱码
3. **`with` 语句**：自动管理文件关闭，即使发生异常也会安全关闭文件
4. **`\n` 换行符**：`f.write()` 不会自动加换行，需要手动添加
5. **空内容判断**：`if not content:` 检查空字符串，避免写入空白日记
6. **`datetime.now().strftime()`**：灵活的时间格式化，`%Y-%m-%d %H:%M:%S` 是最常用的格式
