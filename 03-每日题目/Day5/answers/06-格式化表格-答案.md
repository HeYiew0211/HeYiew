# 06 - 格式化表格 - 答案

## 完整代码

```python
def print_table(data):
    """将 list of dict 数据格式化打印为对齐表格
    
    参数:
        data: list of dict，如 [{"姓名": "张三", "语文": "90"}, ...]
    """
    if not data:
        print("（无数据）")
        return
    
    # 1. 获取所有列名（取第一个字典的 keys）
    keys = list(data[0].keys())
    
    # 2. 计算每列的最大宽度
    # 列宽 = max(列名长度, 该列所有数据中的最大长度)
    col_widths = {}
    for key in keys:
        # 列名本身的宽度
        max_width = len(key)
        # 检查该列所有数据的宽度
        for row in data:
            value_len = len(str(row[key]))
            if value_len > max_width:
                max_width = value_len
        col_widths[key] = max_width
    
    # 3. 打印表头
    header_parts = []
    for key in keys:
        # :<{width} 表示左对齐，宽度为 width
        header_parts.append(f"{key:<{col_widths[key]}}")
    header_line = "  ".join(header_parts)  # 两列之间用两个空格分隔
    print(header_line)
    
    # 4. 打印分隔线
    total_width = sum(col_widths.values()) + 2 * (len(keys) - 1)
    print("-" * total_width)
    
    # 5. 逐行打印数据
    for row in data:
        row_parts = []
        for key in keys:
            value = str(row[key])
            row_parts.append(f"{value:<{col_widths[key]}}")
        print("  ".join(row_parts))
```

## 关键要点

1. **自适应列宽**：`max(len(key), max(len(str(row[key]))))` 确保每列宽度足够容纳所有内容
2. **f-string 对齐语法**：`f"{value:<{width}}"` — `<` 左对齐，`>` 右对齐，`^` 居中对齐，`{width}` 是动态宽度
3. **`str(row[key])`**：先转为字符串再计算长度，兼容数字类型的值
4. **列间分隔**：`"  ".join()` 用两个空格连接各列，使表格更易读
5. **分隔线**：总宽度 = 各列宽度之和 + 列间空格数
6. **扩展性**：该函数适用于任意列名和列数的数据，不需要硬编码
